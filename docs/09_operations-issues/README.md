这里用于记录金蝶运行过程中的问题、排查过程和解决方案。

## 问题分类

- 财务问题
- 供应链问题
- 制造问题
- BOS/二开问题
- 接口问题
- 性能问题
- 权限问题
- 环境和发布问题

## 记录要求

每个问题尽量包含：

- 现象
- 报错原文
- 影响范围
- 产品版本
- 环境信息
- 涉及组织、账簿和期间
- 排查过程
- 根因
- 解决方案
- 预防措施

优先使用 [问题处理模板](../98_templates/issue-resolution.md)。
