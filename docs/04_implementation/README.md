这里用于沉淀金蝶实施交付方法。

## 推荐内容

- [金蝶实施交付作战手册](implementation-delivery-playbook.md)
- 项目调研：业务现状、痛点、组织、流程、单据、报表。
- 蓝图设计：未来流程、系统配置、权限、数据、二开和接口。
- 主数据方案：物料、客户、供应商、部门、仓库、科目。
- 初始化方案：总账、应收应付、库存、资产和历史数据。
- 测试计划：单元测试、集成测试、UAT、月结演练。
- 上线切换：冻结、导入、验证、开放、应急。
- 培训资料：岗位手册、操作步骤、常见问题。
- 验收清单：交付物、测试报告、上线报告和运维移交。
- 风险和问题清单：持续跟踪、闭环和复盘。

## 财务实施重点

- 核算组织和账簿设计。
- 科目表和辅助核算设计。
- 应收应付期初明细。
- 固定资产初始化卡片。
- 月结流程和职责分工。
- 报表公式和管理口径。
