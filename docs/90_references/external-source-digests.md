## 使用方法

1. 先看对应模块的教程页，理解业务和实施逻辑。
2. 再看精读页，了解官方资料中的能力边界和配置对象。
3. 需要核对菜单、字段、参数时，打开精读页里的原始链接。
4. 发现某个外部资料特别有价值时，把它改写成知识页，不直接复制大段原文。
