组织间结算用于处理集团内部多个业务组织、库存组织、核算组织之间的交易。典型场景是 A 公司采购、B 工厂生产、C 公司销售，或者一个组织把货调拨给另一个组织使用。组织间业务如果只做库存调拨，不做结算和财务确认，集团内部收入、成本、往来和库存口径都会混乱。

## 一、适用场景

- 跨法人销售和采购。
- 集团内部调拨。
- 共享采购中心替下属公司采购。
- 工厂生产后调拨给销售公司销售。
- 委托加工或内部委外。
- 多组织之间需要形成内部应收应付、内部收入成本。

## 二、核心概念

| 概念 | 说明 |
| --- | --- |
| 业务组织 | 发起采购、销售、库存业务的组织 |
| 库存组织 | 管理库存数量和仓库的组织 |
| 核算组织 | 进行会计核算和出具账表的组织 |
| 结算组织 | 参与内部结算的买卖双方 |
| 组织间交易 | 集团内部两个组织之间的买卖或调拨 |
| 组织间结算单 | 用于确认内部结算金额、税额和往来的单据 |
| 内部交易抵消 | 集团合并报表时抵消内部收入、成本和往来 |

## 三、业务模式

### 模式 1：跨组织调拨后结算

```text
A 组织库存调拨出库
  -> B 组织调拨入库
  -> 生成组织间结算
  -> A 组织确认内部收入或成本转出
  -> B 组织确认存货和内部应付
```

适合集团内部货物流转明确、需要按内部价格结算的场景。

### 模式 2：销售组织和库存组织不同

```text
销售公司接单
  -> 工厂或仓储组织发货
  -> 外部客户应收确认
  -> 销售公司和工厂之间内部结算
```

适合销售公司负责市场，工厂负责生产发货的集团模式。

### 模式 3：采购中心集中采购

```text
采购中心下采购订单
  -> 供应商送货到下属工厂
  -> 下属工厂入库
  -> 采购中心和下属工厂内部结算
```

适合集团集中采购、下属组织使用物料的场景。

## 四、上线前设计

### 组织关系

要先画清楚组织关系：

| 项目 | 需要确认 |
| --- | --- |
| 哪些组织独立核算 | 是否法人、利润中心或成本中心 |
| 哪些组织有库存 | 仓库归属哪个库存组织 |
| 哪些组织对外交易 | 谁签合同、谁开票、谁收付款 |
| 内部交易双方 | 卖方组织、买方组织 |
| 内部价格 | 成本价、协议价、加成价还是结算价目表 |

### 结算规则

| 规则 | 设计要点 |
| --- | --- |
| 结算时点 | 出库后、入库后、月末集中结算 |
| 价格来源 | 成本价、内部价目表、外部销售价比例 |
| 税务处理 | 是否开票、是否含税、税率 |
| 往来确认 | 内部应收、内部应付科目 |
| 差异处理 | 内部价和成本差异如何入账 |
| 抵消要求 | 集团报表是否需要抵消内部交易 |

## 五、基础配置

组织间结算通常需要配置：

- 组织架构和组织关系。
- 业务组织、库存组织、核算组织映射。
- 内部客户和内部供应商。
- 组织间结算关系。
- 内部价目表或结算价格规则。
- 单据转换和结算生成规则。
- 凭证模板和内部往来科目。
- 权限和审批流。

## 六、操作流程

### 跨组织调拨结算

1. 发出组织创建调拨申请或调拨订单。
2. 发出仓库审核调拨出库。
3. 接收组织审核调拨入库。
4. 根据规则生成组织间结算单。
5. 财务确认结算金额和税额。
6. 生成内部应收、内部应付或成本转移凭证。
7. 月末核对双方内部往来。

### 采购中心集中采购结算

1. 采购中心维护供应商和采购价格。
2. 采购中心下达采购订单。
3. 供应商送货到使用组织仓库。
4. 使用组织审核采购入库。
5. 对外供应商应付由采购中心或指定组织确认。
6. 采购中心和使用组织生成内部结算。
7. 双方内部往来对账。

## 七、财务处理

### 常见凭证方向

发出方：

```text
借：内部应收
  贷：内部收入 / 存货 / 成本转出
```

接收方：

```text
借：存货 / 成本费用
  贷：内部应付
```

具体科目以企业会计政策和集团核算方案为准。内部交易是否确认收入，要由财务负责人和审计口径确认。

### 对账重点

- 发出方内部应收 = 接收方内部应付。
- 发出方结算数量 = 接收方入库数量。
- 结算金额和内部价格规则一致。
- 内部交易在集团合并报表中可以抵消。

## 八、常见问题

| 问题 | 原因 | 处理 |
| --- | --- | --- |
| 调拨完成但没有结算 | 结算规则未命中、单据状态不满足 | 查组织关系、结算规则、单据状态 |
| 双方往来不一致 | 一方凭证未生成或未过账 | 查凭证生成和过账状态 |
| 内部价格不对 | 价目表、成本价或取价时点错误 | 固定取价规则并回归测试 |
| 集团报表抵消困难 | 内部客户供应商映射不规范 | 统一内部客商编码和科目 |
| 发票和内部结算混淆 | 税务口径没定义 | 财务和税务确认内部交易处理 |

## 九、实际案例

### 案例：工厂调货给销售公司

背景：

- A 工厂生产成品。
- B 销售公司负责对外销售。
- 货物从 A 工厂仓库调拨到 B 销售公司仓库。

方案：

1. A 工厂做调拨出库。
2. B 销售公司做调拨入库。
3. 系统按内部结算价生成组织间结算。
4. A 工厂形成内部应收，B 销售公司形成内部应付。
5. B 销售公司对外销售时确认外部收入和应收。
6. 集团合并时抵消 A 与 B 的内部往来和内部交易。

检查：

- A 出库数量和 B 入库数量一致。
- 内部结算价已审核。
- 内部应收应付双方一致。
- 外部销售成本使用正确存货成本。
