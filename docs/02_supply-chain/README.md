## 后续继续加厚方向

- 采购管理详细教程：采购申请、采购订单、采购入库、暂估、应付。
- 销售管理详细教程：报价、订单、发货、出库、退货、信用。
- 库存管理详细教程：即时库存、批号、库位、盘点、调拨。
- 存货核算详细教程：计价方法、出库成本、凭证和对账。
- 组织间结算、条码管理、供应商协同。

## 和财务的关系

- 采购业务通常影响应付、存货和成本。
- 销售业务通常影响应收、收入和库存。
- 库存关账、存货核算和成本核算会影响总账月结。
