存货核算负责把库存数量变成财务金额。库存管理回答“有多少”，存货核算回答“值多少钱、成本是多少、凭证怎么进总账”。

## 一、存货核算大图

```text
采购入库/其他入库/完工入库
  -> 入库成本确认
  -> 销售出库/生产领料/其他出库
  -> 出库成本计算
  -> 成本调整
  -> 存货凭证
  -> 总账对账
```

## 二、存货核算对象

| 对象 | 作用 |
| --- | --- |
| 存货组织 | 成本核算主体 |
| 物料 | 成本对象 |
| 仓库 | 库存地点 |
| 批号 | 批次成本追踪 |
| 库存单据 | 成本来源和成本消耗 |
| 计价方法 | 出库成本计算规则 |
| 存货科目 | 总账科目 |

## 三、计价方法

| 方法 | 说明 | 适用 |
| --- | --- | --- |
| 移动平均 | 每次入库后重新计算平均成本 | 常规贸易、库存周转快 |
| 月加权平均 | 月末按全月入库和期初计算平均成本 | 月末统一核算 |
| 先进先出 | 先入库批次先出库 | 批次明显、价格波动 |
| 个别计价 | 每件或每批单独成本 | 贵重或定制产品 |
| 标准成本 | 按标准成本出库，差异单独分析 | 制造企业、成本控制 |

计价方法确定后不要轻易变更。变更会影响存货成本、主营业务成本和报表。

## 四、入库成本

### 采购入库成本

采购入库成本可能来自：

- 采购订单价格。
- 采购发票价格。
- 暂估价格。
- 费用分摊。
- 关税、运费、保险费等采购附加成本。

检查点：

- 入库数量正确。
- 入库单价正确。
- 税额是否计入成本要按税务口径。
- 采购费用是否需要分摊。
- 发票到达后是否调整暂估成本。

### 完工入库成本

完工入库成本来自制造成本：

- 直接材料。
- 直接人工。
- 制造费用。
- 委外加工费。
- 成本调整。

如果制造成本没有算完，完工入库成本可能不准确。

### 其他入库成本

其他入库常见于盘盈、赠品、调整。

财务要确认：

- 是否有合理估价。
- 入哪个存货科目。
- 对方科目是什么。
- 是否影响损益。

## 五、出库成本

### 销售出库成本

销售出库需要结转主营业务成本。

```text
借：主营业务成本
贷：库存商品
```

检查：

- 销售出库单是否审核。
- 出库成本是否计算。
- 是否存在负库存。
- 是否有成本为 0 的出库。
- 成本凭证是否生成。

### 生产领料成本

生产领料进入生产成本。

```text
借：生产成本-直接材料
贷：原材料
```

检查：

- 领料是否关联生产订单。
- 材料成本是否正确。
- 超领和补料是否审批。
- 退料是否冲回成本。

### 其他出库成本

其他出库可能进入费用、损失或其他业务成本。

要确认：

- 出库原因。
- 对方科目。
- 是否审批。
- 是否需要税务处理。

## 六、成本调整

成本调整用于修正存货成本。

常见场景：

- 发票价格和暂估价格差异。
- 采购费用分摊。
- 成本计算异常。
- 期初成本错误。
- 盘盈盘亏成本调整。

调整原则：

- 调整原因必须清楚。
- 调整金额必须可追溯。
- 大额调整必须审批。
- 调整后要重新对账。

## 七、存货凭证

存货核算完成后，需要生成凭证进入总账。

常见凭证：

| 业务 | 分录 |
| --- | --- |
| 采购入库 | 借：原材料/库存商品，贷：应付暂估/应付账款 |
| 销售出库 | 借：主营业务成本，贷：库存商品 |
| 生产领料 | 借：生产成本，贷：原材料 |
| 完工入库 | 借：库存商品，贷：生产成本 |
| 盘盈 | 借：存货，贷：待处理财产损溢等 |
| 盘亏 | 借：待处理财产损溢等，贷：存货 |

凭证检查：

- 科目是否按物料或存货类别映射。
- 金额是否等于核算结果。
- 辅助核算是否完整。
- 凭证是否审核过账。
- 能否反查库存单据。

## 八、月末存货核算顺序

```text
清理库存单据
  -> 检查负库存
  -> 入库核算
  -> 出库核算
  -> 成本调整
  -> 生成存货凭证
  -> 存货与总账对账
  -> 库存结账
```

月末必须先保证库存单据完整，否则成本会不准。

## 九、存货对账

### 对账关系

| 对账 | 标准 |
| --- | --- |
| 库存数量 | 即时库存 = 库存明细账 |
| 存货金额 | 存货核算余额 = 存货报表 |
| 总账金额 | 存货核算 = 总账存货科目 |
| 销售成本 | 销售出库成本 = 主营业务成本 |
| 生产成本 | 生产订单成本 = 生产成本科目 |

### 差异分类

- 库存单据未审核。
- 存货核算未完成。
- 存货凭证未生成。
- 凭证未过账。
- 总账手工调整存货科目。
- 期初不一致。
- 查询条件不一致。

## 十、常见问题

### 出库成本为 0

可能原因：

- 期初无成本。
- 入库未核算。
- 负库存出库。
- 计价方法设置错误。
- 物料成本对象异常。

处理：

1. 查出库前是否有库存成本。
2. 查入库核算是否完成。
3. 查是否负库存。
4. 重新核算出库成本。
5. 生成或调整凭证。

### 存货和总账不一致

排查：

1. 固定组织、期间、币别、科目。
2. 查存货核算余额。
3. 查存货凭证生成情况。
4. 查凭证是否过账。
5. 查总账是否有手工存货凭证。
6. 查期初余额。

### 暂估长期不冲

排查：

1. 查采购入库暂估清单。
2. 查发票是否到达。
3. 查应付单是否生成。
4. 查冲暂估凭证。
5. 查供应商对账。

## 十一、存货核算实施验收清单

- [ ] 存货组织和库存组织关系确认。
- [ ] 计价方法确认。
- [ ] 物料存货科目映射完整。
- [ ] 采购入库成本能确认。
- [ ] 销售出库成本能计算。
- [ ] 生产领料和完工成本能衔接。
- [ ] 成本调整流程可用。
- [ ] 存货凭证能生成并过账。
- [ ] 存货核算和总账能对账。
- [ ] 月末存货结账流程跑通。
