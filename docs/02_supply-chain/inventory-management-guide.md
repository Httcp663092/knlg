库存管理负责“东西在哪里、有多少、能不能用、怎么流转”。库存管理主要管数量和状态，存货核算主要管金额和成本。两者不能混淆。

## 一、库存管理大图

```text
物料和仓库基础资料
  -> 入库
  -> 出库
  -> 调拨
  -> 盘点
  -> 批号/序列号/库位管理
  -> 即时库存
  -> 库存账表
  -> 存货核算
```

## 二、库存基础对象

| 对象 | 作用 | 示例 |
| --- | --- | --- |
| 库存组织 | 库存归属主体 | A 工厂 |
| 仓库 | 物理或逻辑存放地点 | 原材料仓、成品仓 |
| 库位 | 仓库内位置 | A-01-01 |
| 物料 | 库存对象 | M100 原材料 |
| 批号 | 批次追踪 | 20260601A |
| 序列号 | 单件追踪 | SN001 |
| 库存状态 | 是否可用 | 可用、冻结、待检 |
| 即时库存 | 当前库存数量 | 可用量、锁定量 |

## 三、库存单据类型

| 单据 | 业务场景 | 数量影响 |
| --- | --- | --- |
| 采购入库 | 外购到货入库 | 增加 |
| 销售出库 | 发给客户 | 减少 |
| 生产领料 | 生产耗用 | 减少 |
| 完工入库 | 成品入库 | 增加 |
| 其他入库 | 盘盈、赠品、调整 | 增加 |
| 其他出库 | 盘亏、报废、领用 | 减少 |
| 调拨单 | 仓库转移 | 一边减一边增 |
| 盘点单 | 实物盘点调整 | 按盈亏调整 |
| 形态转换 | 物料状态或形态变化 | 视业务调整 |

## 四、入库管理

### 入库来源

- 采购入库。
- 生产完工入库。
- 委外入库。
- 销售退货入库。
- 其他入库。

### 入库操作步骤

1. 选择入库类型。
2. 选择库存组织。
3. 选择仓库和库位。
4. 选择物料。
5. 录入数量和单位。
6. 录入批号、序列号、生产日期、有效期。
7. 保存、提交、审核。
8. 查询即时库存。

### 入库检查

- 单据是否审核后才更新库存。
- 仓库是否正确。
- 库位是否必填。
- 批号是否符合规则。
- 质检状态是否正确。
- 是否允许无来源入库。

## 五、出库管理

### 出库来源

- 销售出库。
- 生产领料。
- 委外发料。
- 其他出库。
- 调拨出库。
- 盘亏出库。

### 出库操作步骤

1. 选择出库类型。
2. 选择库存组织。
3. 选择仓库和库位。
4. 选择物料、批号、序列号。
5. 检查可用库存。
6. 录入出库数量。
7. 保存、提交、审核。
8. 查询即时库存减少。

### 出库检查

- 是否允许负库存。
- 可用库存是否足够。
- 批号和库位是否正确。
- 出库单位和库存单位换算是否正确。
- 是否超订单、超领料。
- 是否需要审批。

## 六、调拨管理

调拨用于仓库、库存组织或法人之间的库存转移。

### 调拨类型

| 类型 | 说明 | 财务影响 |
| --- | --- | --- |
| 同仓库内库位调拨 | 位置变化 | 通常无财务影响 |
| 同组织跨仓调拨 | 仓库变化 | 通常无财务影响 |
| 跨库存组织调拨 | 组织变化 | 可能影响内部结算 |
| 跨法人调拨 | 法人变化 | 通常需要内部交易 |

### 调拨检查

- 调出仓库和调入仓库是否正确。
- 在途库存是否管理。
- 是否需要调拨出库和调拨入库两步。
- 跨组织是否需要组织间结算。
- 调拨成本如何确定。

## 七、盘点管理

盘点用于让系统库存和实物库存一致。

### 盘点流程

```text
盘点计划
  -> 生成盘点表
  -> 冻结或控制库存业务
  -> 实物盘点
  -> 录入盘点数量
  -> 复盘
  -> 生成盘盈盘亏
  -> 财务确认
```

### 盘点检查

- 是否按仓库、库位、物料、批号盘点。
- 盘点期间是否暂停出入库。
- 盘盈盘亏原因是否说明。
- 大额差异是否复盘。
- 盘盈盘亏是否生成库存调整。
- 财务是否确认金额和科目。

## 八、批号、序列号和保质期

### 批号管理

适合：

- 食品。
- 药品。
- 化工。
- 电子元器件。
- 需要追溯的原材料。

批号必须贯穿采购入库、生产领料、销售出库、退货和盘点。

### 序列号管理

适合单件价值高、需要单件追踪的产品，例如设备、整机、仪器。

序列号常见问题：

- 入库时未录序列号。
- 出库时序列号和实物不一致。
- 退货时找不到原序列号。
- 序列号重复。

### 保质期管理

检查：

- 生产日期。
- 有效期至。
- 是否近效期预警。
- 是否先进先出。
- 过期品是否冻结。

## 九、库存状态

常见库存状态：

| 状态 | 说明 |
| --- | --- |
| 可用 | 可销售、可领用 |
| 待检 | 需质检，暂不可用 |
| 不合格 | 不可正常使用 |
| 冻结 | 管理冻结 |
| 在途 | 调拨或运输中 |

库存状态会影响可用量。查库存时要区分账面数量和可用数量。

## 十、库存和财务的关系

库存管理主要管数量，财务关注金额。财务不会只看即时库存，而是要看存货核算和总账。

```text
库存单据审核
  -> 库存数量变化
  -> 存货核算确定金额
  -> 生成存货凭证
  -> 总账存货科目变化
```

对账：

- 即时库存对库存明细账。
- 库存明细账对存货核算。
- 存货核算对总账。

## 十一、库存月结

月结前要清理：

- 未审核入库单。
- 未审核出库单。
- 未完成调拨。
- 未完成盘点。
- 负库存。
- 批号或库位异常。
- 存货核算未完成。

库存月结后，一般不允许随意修改历史期间库存业务。

## 十二、常见问题

### 即时库存和实物不一致

排查：

1. 查仓库、库位、批号是否一致。
2. 查本期出入库明细。
3. 查是否有未审核单据。
4. 查是否有盘点未处理。
5. 查是否有调拨在途。
6. 组织实物复盘。

### 出库提示库存不足

可能原因：

- 查错仓库。
- 批号库存不足。
- 库存状态不可用。
- 有锁定或预留。
- 单位换算错误。
- 即时库存未更新。

### 负库存怎么处理

负库存是高风险状态。处理：

1. 查是否允许负库存参数。
2. 查负库存产生单据。
3. 补录正确入库或调整错误出库。
4. 重新做存货核算。
5. 检查总账成本影响。

## 十三、库存实施验收清单

- [ ] 仓库、库位、库存组织配置正确。
- [ ] 采购入库、销售出库、生产领料、完工入库跑通。
- [ ] 批号、序列号按需启用并可追溯。
- [ ] 调拨流程跑通。
- [ ] 盘点流程跑通。
- [ ] 库存状态和可用量逻辑清楚。
- [ ] 即时库存和库存明细账一致。
- [ ] 存货核算和总账衔接清楚。
