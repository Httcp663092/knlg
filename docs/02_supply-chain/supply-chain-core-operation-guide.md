这篇用于建立金蝶供应链主流程。供应链不是只管采购、销售、库存单据，它会直接影响应收、应付、存货、成本和总账月结。学习供应链时要同时看“业务流、库存流、资金流、凭证流”。

## 一、供应链总体地图

```text
基础资料
  -> 采购业务
  -> 库存收发
  -> 销售业务
  -> 存货核算
  -> 应收应付
  -> 总账凭证
  -> 报表和月结
```

| 模块 | 关键单据 | 输出 | 财务影响 |
| --- | --- | --- | --- |
| 采购管理 | 采购申请、采购订单、采购入库、采购退料 | 到货、入库、供应商履约 | 应付、存货、暂估 |
| 销售管理 | 销售订单、发货通知、销售出库、销售退货 | 发货、退货、客户履约 | 应收、收入、成本 |
| 库存管理 | 其他入库、其他出库、调拨、盘点 | 库存数量和库位 | 存货数量、盘盈盘亏 |
| 存货核算 | 入库核算、出库核算、成本调整 | 存货金额和成本 | 存货、主营业务成本 |
| 组织间结算 | 跨组织调拨、内部交易 | 内部结算单据 | 内部应收应付、抵消 |

## 二、上线前必须确认的基础资料

供应链上线失败，很多不是流程问题，而是基础资料口径不稳定。

| 基础资料 | 关键字段 | 检查点 |
| --- | --- | --- |
| 物料 | 编码、名称、规格、基本单位、库存单位、采购/销售/生产属性 | 是否重复、单位是否正确、是否启用批号 |
| 供应商 | 编码、结算组织、付款条件、税率、银行账号 | 是否重复，是否经过准入 |
| 客户 | 编码、结算客户、信用额度、收款条件、税率 | 收货客户和结算客户是否区分 |
| 仓库 | 所属组织、仓库属性、是否启用库位 | 不同组织仓库不能混用 |
| 库位 | 仓库下的物理位置 | 适合精细库存或条码管理 |
| 批号 | 批次规则、保质期 | 食品、医药、电子件常用 |
| 计量单位 | 基本单位、采购单位、销售单位、换算率 | 换算率错会导致数量异常 |
| 价格资料 | 采购价目表、销售价目表、折扣 | 价格审批和有效期要明确 |

## 三、采购到付款流程

### 标准流程

```text
采购申请
  -> 采购订单
  -> 采购到货
  -> 采购入库
  -> 应付单
  -> 付款单
  -> 应付核销
  -> 生成凭证
```

不同企业可能没有采购申请或到货环节，但采购订单、入库、应付、付款这条主线要清楚。

### 操作步骤：采购订单

1. 选择采购组织和采购员。
2. 选择供应商。
3. 录入物料、数量、单位、含税单价、税率。
4. 确认交货日期、交货地点和仓库。
5. 检查付款条件和结算方式。
6. 保存、提交、审核。

检查点：

- 供应商是否正确。
- 采购组织和库存组织是否匹配。
- 物料是否允许采购。
- 单位换算是否正确。
- 价格是否来自有效价目表或审批。

### 操作步骤：采购入库

1. 从采购订单或到货单下推采购入库单。
2. 选择收货仓库和库位。
3. 核对实收数量、批号、生产日期、保质期。
4. 处理不合格数量或退料数量。
5. 保存、审核入库单。
6. 查询即时库存确认数量增加。

检查点：

- 入库组织和仓库是否正确。
- 实收数量是否等于实际收货。
- 批号和保质期是否完整。
- 是否需要质检。
- 是否允许超订单入库。

### 操作步骤：应付确认和付款

1. 根据采购入库或采购发票生成应付单。
2. 检查供应商、结算供应商、币别、税率、价税合计。
3. 审核应付单。
4. 生成应付凭证。
5. 根据付款计划生成付款单。
6. 审核付款单并生成付款凭证。
7. 付款单和应付单核销。
8. 查询供应商应付余额。

### 暂估应付

如果货已入库但发票未到，企业通常需要暂估。

```text
采购入库
  -> 暂估应付
  -> 暂估凭证
  -> 发票到达
  -> 冲暂估
  -> 正式应付
```

暂估管理重点：

- 暂估口径按入库还是到票确认，要在蓝图中明确。
- 月末暂估清单要能输出。
- 发票到达后要能冲回暂估。
- 暂估科目和正式应付科目要设计清楚。

## 四、销售到收款流程

### 标准流程

```text
销售报价
  -> 销售订单
  -> 发货通知
  -> 销售出库
  -> 应收单
  -> 收款单
  -> 应收核销
  -> 生成凭证
```

### 操作步骤：销售订单

1. 选择销售组织和销售员。
2. 选择客户、收货客户、结算客户。
3. 录入物料、数量、单位、含税单价、税率。
4. 确认交货日期、发货仓库、收货地址。
5. 检查信用额度、价格政策和审批状态。
6. 保存、提交、审核。

检查点：

- 客户和结算客户是否正确。
- 是否超信用额度。
- 价格是否有效。
- 销售单位和库存单位换算是否正确。
- 是否需要预收款控制。

### 操作步骤：销售出库

1. 根据销售订单或发货通知下推销售出库单。
2. 选择仓库和库位。
3. 检查可用库存。
4. 录入实际出库数量、批号、序列号。
5. 审核出库单。
6. 查询即时库存确认减少。

### 操作步骤：应收和收款

1. 根据销售出库、销售发票或订单确认口径生成应收单。
2. 审核应收单。
3. 生成应收凭证。
4. 根据银行到账录入收款单。
5. 审核收款单并生成收款凭证。
6. 收款单和应收单核销。
7. 查询客户余额和账龄。

### 销售退货

退货要同时关注库存、应收和收入成本。

```text
退货申请
  -> 销售退货单
  -> 退货入库
  -> 红字应收或退款
  -> 成本冲回
```

检查点：

- 退货是否关联原销售订单或出库单。
- 是否影响客户账款。
- 是否需要退款。
- 退货入库成本怎么确定。
- 报表中收入、成本是否冲回。

## 五、库存管理

### 库存核心对象

| 对象 | 作用 |
| --- | --- |
| 库存组织 | 管理库存归属 |
| 仓库 | 库存存放地点 |
| 库位 | 仓库下的具体位置 |
| 批号 | 同一批次物料的追踪口径 |
| 序列号 | 单件物品追踪 |
| 即时库存 | 当前可查询库存数量 |
| 库存状态 | 可用、冻结、待检、不合格等 |

### 常见库存单据

| 单据 | 适用场景 | 财务影响 |
| --- | --- | --- |
| 采购入库 | 外购入库 | 增加库存，后续形成应付 |
| 销售出库 | 销售发货 | 减少库存，后续结转成本 |
| 生产领料 | 生产耗用材料 | 减少材料，进入生产成本 |
| 产品入库 | 完工入库 | 增加产成品 |
| 其他入库 | 盘盈、赠品、调整 | 可能形成营业外或库存调整 |
| 其他出库 | 盘亏、报废、领用 | 可能形成费用或损失 |
| 调拨单 | 仓库或组织间转移 | 同组织不影响总额，跨组织可能结算 |
| 盘点单 | 盘盈盘亏 | 调整库存数量和金额 |

### 库存盘点流程

```text
制定盘点计划
  -> 冻结或控制库存业务
  -> 生成盘点表
  -> 实物盘点
  -> 录入盘点数量
  -> 生成盘盈盘亏单
  -> 审核库存调整
  -> 财务确认
```

盘点检查：

- 盘点范围是否明确。
- 盘点期间是否停止出入库。
- 批号、库位、序列号是否盘到。
- 盘盈盘亏是否有原因说明。
- 财务是否确认盘盈盘亏科目。

## 六、存货核算和财务衔接

### 存货核算为什么重要

库存管理主要管数量，存货核算管金额。销售出库后，如果成本没有核算，财务的主营业务成本就不完整。

常见成本计价方式：

| 方式 | 说明 | 注意 |
| --- | --- | --- |
| 移动平均 | 每次入库后更新平均成本 | 常用于普通存货 |
| 月加权平均 | 月末按总金额总数量计算 | 月末核算依赖完整单据 |
| 先进先出 | 按先入库先出库成本 | 批次和成本层要清晰 |
| 标准成本 | 按预设标准成本 | 差异要单独分析 |

### 月末供应链到总账顺序

```text
采购、销售、库存单据审核完成
  -> 存货核算
  -> 出库成本计算
  -> 成本调整
  -> 生成存货凭证
  -> 凭证审核过账
  -> 库存、存货、总账对账
```

### 对账重点

| 对账 | 目标 |
| --- | --- |
| 即时库存对库存台账 | 数量一致 |
| 库存台账对存货核算 | 数量和金额一致 |
| 存货核算对总账 | 存货科目余额一致 |
| 销售出库对成本 | 已出库业务都有成本 |
| 采购入库对应付 | 入库和应付口径能解释 |

## 七、常见问题排查

### 库存数量不对

排查顺序：

1. 固定库存组织、仓库、物料、批号、库位。
2. 查即时库存。
3. 查库存明细账。
4. 查本期所有出入库单据。
5. 查是否有未审核或反审核单据。
6. 查是否有调拨、盘点、其他出入库。

### 销售订单不能出库

可能原因：

- 库存不足。
- 物料状态不可用。
- 信用控制未通过。
- 订单未审核。
- 仓库或库存组织不一致。
- 批号或序列号未指定。

### 采购入库不能生成应付

可能原因：

- 应付确认口径不是按入库。
- 供应商或结算供应商缺失。
- 税率或价格异常。
- 单据状态未审核。
- 采购组织和结算组织配置不完整。

### 存货和总账不一致

排查顺序：

1. 查存货核算是否完成。
2. 查存货凭证是否生成。
3. 查凭证是否过账。
4. 查总账是否有手工存货凭证。
5. 查期初是否一致。
6. 查查询条件是否一致。

## 八、供应链实施验收清单

- [ ] 物料、客户、供应商、仓库、计量单位已确认。
- [ ] 采购订单到入库到应付流程跑通。
- [ ] 付款和应付核销流程跑通。
- [ ] 销售订单到出库到应收流程跑通。
- [ ] 收款和应收核销流程跑通。
- [ ] 库存盘点流程跑通。
- [ ] 存货核算和出库成本计算跑通。
- [ ] 存货凭证能生成并过账。
- [ ] 库存、存货核算、总账能对账。
- [ ] 月结顺序和责任人明确。
