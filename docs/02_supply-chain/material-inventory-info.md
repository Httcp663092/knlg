来源：金蝶开发者官网产品页中的“物料库存信息”。

## 功能定位

物料库存信息是在物料主数据基础上定义库存管理策略的资料。它决定物料在库存单据中如何被管理，例如库存单位、批号、序列号、保质期等。

## 应用场景

企业导入系统初期，需要先准备基础资料。只有维护了物料库存信息后，库存单据才能录入相关物料；如果启用了批号管理、保质期管理等属性，后续库存单据才能维护批号、保质期相关字段。

## 系统路径

| 版本 | 路径 |
| --- | --- |
| V6.0 及以上 | 基础服务云 -> 基础资料 -> 供应链数据 -> 物料库存信息 |
| V6.0 以下 | 系统服务云 -> 基础资料 -> 供应链数据 -> 物料库存信息 |

## 关键字段理解

| 字段 | 作用 | 影响 |
| --- | --- | --- |
| 物料编码 | 关联物料主数据 | 决定这条库存策略属于哪个物料 |
| 物料创建组织 | 物料主数据创建组织 | 影响组织隔离和资料分配 |
| 库存单位 | 库存业务使用的单位 | 影响库存数量、换算和报表 |
| 批号管理 | 是否按批次管理库存 | 启用后入库、出库、盘点通常需要批号 |
| 序列号管理 | 是否按单件序列号管理 | 适合设备、整机、唯一件追踪 |
| 保质期管理 | 是否管理有效期 | 启用后需要维护生产日期、有效期或到期日 |

## 启用批号管理的影响

启用批号管理后，业务上会带来这些变化：

- 入库时需要录入或生成批号。
- 出库时需要指定批号。
- 库存余额会按物料、仓库、批号等维度拆分。
- 盘点时需要按批号盘点。
- 追溯时可以按批号查询来源和去向。

适合启用批号管理的物料：

- 食品、药品、化工品。
- 原材料批次差异明显的物料。
- 质量追溯要求高的物料。
- 客户或法规要求按批次追溯的物料。

## 启用序列号管理的影响

序列号管理通常用于单件追踪。启用后，系统更关注每一个具体实物。

适用场景：

- 设备。
- 整机。
- 高价值部件。
- 售后需要追踪唯一件的产品。

实施注意：

- 序列号规则要提前设计。
- 入库、出库、调拨、维修等业务都要考虑序列号流转。
- 如果历史库存没有序列号，上线切换要制定补录方案。

## 启用保质期管理的影响

启用保质期管理后，库存业务要关注生产日期、有效期、到期日。

常见控制：

- 入库录入生产日期或到期日。
- 出库按先进先出或近效期优先。
- 库存报表可按到期预警查询。
- 盘点和报废要关注过期物料。

## 上线配置步骤

1. 确认哪些物料需要库存管理。
2. 维护物料主数据。
3. 进入物料库存信息路径。
4. 按组织维护库存单位。
5. 判断是否启用批号、序列号、保质期。
6. 用典型库存单据验证入库、出库、调拨、盘点。
7. 检查库存报表是否按预期展示。

## 财务和供应链影响

- 库存单位会影响库存数量和存货核算数量。
- 批号、序列号、保质期会影响库存维度，进而影响库存报表查询。
- 物料库存信息配置错误，可能导致业务单据无法录入物料，或后期追溯困难。

## 常见问题

### 库存单据选不到物料

排查：

1. 物料主数据是否已审核、启用。
2. 物料库存信息是否已维护。
3. 物料是否分配到当前组织。
4. 库存组织和使用组织是否匹配。

### 启用批号后单据保存失败

排查：

1. 当前物料是否启用批号管理。
2. 单据上是否录入批号。
3. 批号规则是否配置。
4. 批号是否已存在或是否允许手工录入。

### 保质期字段不出现

排查：

1. 物料库存信息是否启用保质期管理。
2. 单据页面是否显示相关字段。
3. 单据类型是否支持保质期。
4. 权限或页面扩展是否隐藏字段。
