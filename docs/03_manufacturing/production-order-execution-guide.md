生产订单是车间执行生产任务的核心单据。金蝶官方生产订单说明里，生产订单由计划部门下达，要求生产车间执行，车间根据生产订单领料、组织生产、调度资源，并支持普通生产、返工、受托加工、联副产品、无 BOM、产品配置等业务。

## 一、生产订单流程

```text
计划订单或手工创建
  -> 生产订单
  -> 生产用料清单
  -> 备料
  -> 领料/倒冲
  -> 工序汇报
  -> 完工入库
  -> 退料/补料
  -> 订单结案
  -> 成本核算
```

## 二、生产订单前置条件

必须准备：

- 物料资料。
- BOM。
- 生产车间。
- 工厂日历。
- 单据类型。
- 仓库和发料规则。
- 工艺路线和工作中心。
- 权限和审批流。

如果这些资料缺失，生产订单可能能建但执行会出问题。

## 三、生产订单关键字段

| 字段 | 说明 |
| --- | --- |
| 生产组织 | 哪个组织生产 |
| 生产车间 | 哪个车间执行 |
| 产品编码 | 要生产的成品或半成品 |
| 生产数量 | 计划生产数量 |
| 计划开工日期 | 预计开始 |
| 计划完工日期 | 预计完成 |
| BOM 版本 | 用哪套产品结构 |
| 工艺路线 | 用哪套生产工艺 |
| 单据类型 | 普通生产、返工、受托加工等 |
| 状态 | 创建、审核、开工、完工、结案 |

## 四、创建生产订单

来源：

- MRP 计划订单投放。
- 销售订单直接下推。
- 手工创建。
- 返工业务创建。
- 委外或特殊业务转换。

操作步骤：

1. 选择生产组织和生产车间。
2. 选择产品物料。
3. 录入生产数量。
4. 确认 BOM 版本。
5. 确认计划开工和完工日期。
6. 确认入库仓库。
7. 保存、提交、审核。
8. 检查生产用料清单是否生成。

## 五、生产用料清单

生产用料清单指定生产订单需要使用的半成品和原材料，通常由系统根据 BOM 自动创建。

检查点：

- 子项物料是否完整。
- 用量是否正确。
- 损耗是否带入。
- 发料方式是否正确。
- 仓库和库位是否正确。
- 替代料是否正确。
- 工序用料是否正确。

用料清单可以因为实际生产变化而调整，但调整必须受控，否则成本和计划都会变形。

## 六、备料和缺料检查

生产前要查缺料。

检查方式：

- 按生产订单查用料。
- 查库存可用量。
- 查采购在途。
- 查调拨在途。
- 查替代料。
- 查是否允许欠料开工。

缺料处理：

- 采购补料。
- 调拨补料。
- 使用替代料。
- 调整生产排程。
- 拆分生产订单。

## 七、生产领料

### 领料方式

| 方式 | 说明 | 适用 |
| --- | --- | --- |
| 按单领料 | 根据生产订单逐笔领料 | 管理精细 |
| 配套领料 | 按套料一次领 | 组装类生产 |
| 倒冲领料 | 完工时自动扣料 | BOM 稳定、现场不逐笔领料 |
| 补料 | 异常消耗补领 | 损耗、报废 |
| 退料 | 多领或不用退回 | 订单结束或变更 |

### 领料操作步骤

1. 选择生产订单。
2. 选择用料清单。
3. 生成生产领料单。
4. 选择发料仓库、库位、批号。
5. 审核领料单。
6. 查询库存减少。
7. 后续进入生产成本。

### 领料风险

- 超领导致成本偏高。
- 少领导致完工成本不完整。
- 领错批号导致追溯失败。
- 倒冲依赖 BOM 准确。
- 退料不及时导致订单成本不准。

## 八、工序汇报和质检

如果启用车间工序，生产订单执行中要记录工序。

工序汇报内容：

- 工序。
- 开工时间。
- 完工时间。
- 合格数量。
- 不良数量。
- 返工数量。
- 工时。
- 操作人。

质检结果：

- 合格。
- 不合格。
- 让步接收。
- 返工。
- 报废。

工序数据影响生产进度、质量追溯、人工成本和制造费用分配。

## 九、完工入库

完工入库表示产品完成并进入库存。

操作步骤：

1. 选择生产订单。
2. 录入完工数量。
3. 选择入库仓库和库位。
4. 录入批号或序列号。
5. 处理联产品、副产品。
6. 审核完工入库单。
7. 查询成品库存增加。

检查点：

- 完工数量是否超过生产订单。
- 是否允许部分完工。
- 是否有质检要求。
- 是否存在未领料。
- 是否触发倒冲领料。
- 是否进入成本核算。

## 十、补料、退料和报废

### 补料

补料用于生产异常消耗。

原因：

- 材料损耗。
- 报废重做。
- 工艺变更。
- BOM 漏料。

补料必须审批，避免成为隐藏成本漏洞。

### 退料

退料用于把未用完材料退回仓库。

检查：

- 是否关联原生产订单。
- 是否退回正确仓库。
- 批号是否一致。
- 是否冲减生产成本。

### 报废

报废要记录原因、责任和成本影响。

## 十一、生产订单结案

结案表示生产订单不再继续执行。

结案前检查：

- 领料是否完成。
- 退料是否完成。
- 完工入库是否完成。
- 工序汇报是否完成。
- 质量异常是否处理。
- 成本核算是否需要继续处理。

结案后通常不允许随意继续领料或入库。

## 十二、生产订单和成本

生产订单是成本对象之一。

成本来源：

- 直接材料：领料或倒冲。
- 直接人工：工时或工资分配。
- 制造费用：费用归集和分配。
- 委外加工费：委外应付。
- 废品损失：报废处理。

成本检查：

- 领料金额是否进入生产成本。
- 完工成本是否结转到库存商品。
- 在产品是否合理。
- 生产订单是否有异常余额。

## 十三、常见问题

### 生产订单无法审核

排查：

1. 物料是否允许生产。
2. BOM 是否有效。
3. 生产车间是否维护。
4. 工厂日历是否维护。
5. 单据类型是否可用。
6. 权限和审批流是否正确。

### 领料单生成失败

可能原因：

- 用料清单未审核。
- 库存不足。
- 发料仓库未维护。
- 批号或库位缺失。
- 子项物料不可用。

### 完工入库失败

可能原因：

- 生产订单未审核。
- 未达到完工条件。
- 入库仓库缺失。
- 批号或序列号未录。
- 质检未完成。

### 生产订单成本异常

排查：

1. 查材料领用。
2. 查补料退料。
3. 查完工数量。
4. 查人工制造费用分配。
5. 查成本计算结果。

## 十四、生产执行验收清单

- [ ] 生产订单能从 MRP 投放生成。
- [ ] 生产订单能手工创建。
- [ ] 生产用料清单正确。
- [ ] 缺料检查可用。
- [ ] 生产领料、补料、退料跑通。
- [ ] 工序汇报按需跑通。
- [ ] 完工入库能增加库存。
- [ ] 生产订单能结案。
- [ ] 生产订单成本能计算。
- [ ] 生产执行和总账成本能对账。
