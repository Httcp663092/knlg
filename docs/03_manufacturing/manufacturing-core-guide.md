生产制造模块的核心目标，是把销售需求、预测需求或库存补货需求，转成可执行的生产计划和生产订单，并把材料消耗、人工制造费用、完工入库和成本核算沉淀为可追溯的数据。

## 一、制造业务大图

```text
工程数据
  -> 需求来源
  -> MRP/MPS 计划
  -> 生产订单
  -> 领料
  -> 工序汇报
  -> 完工入库
  -> 成本归集和分配
  -> 总账凭证
```

| 模块 | 关键资料/单据 | 作用 |
| --- | --- | --- |
| 工程数据 | BOM、工艺路线、工作中心、资源 | 决定产品怎么做 |
| 计划管理 | 预测单、销售订单、MRP、计划订单 | 决定做什么、做多少、什么时候做 |
| 生产管理 | 生产订单、领料单、汇报单、完工入库 | 执行生产 |
| 委外管理 | 委外订单、委外发料、委外入库 | 外协加工 |
| 成本核算 | 材料、人工、制造费用、完工成本 | 形成产品成本 |

## 二、工程数据

### 物料生产属性

制造前先确认物料属性：

| 属性 | 含义 | 示例 |
| --- | --- | --- |
| 自制 | 企业内部生产 | 成品、半成品 |
| 外购 | 从供应商采购 | 原材料、辅料 |
| 委外 | 外协加工 | 表面处理件 |
| 虚拟件 | 不实际库存，仅展开 BOM | 套件、组件 |
| 批号管理 | 需要批次追踪 | 食品、电子元件 |
| 序列号管理 | 单件追踪 | 设备、整机 |

### BOM

BOM 是产品结构，表示做一个成品需要哪些材料。

| 字段 | 说明 |
| --- | --- |
| 父项物料 | 要生产的成品或半成品 |
| 子项物料 | 需要消耗的材料 |
| 用量 | 单位产品耗用数量 |
| 损耗率 | 生产损耗 |
| 生效日期 | BOM 有效期 |
| 替代料 | 缺料时可替代的物料 |
| 发料方式 | 按单领料、倒冲、调拨等 |

BOM 检查点：

- 是否有循环 BOM。
- 单位换算是否正确。
- 损耗率是否合理。
- 替代料是否经过工程和财务确认。
- 版本是否生效。
- 子项物料是否可采购或可生产。

### 工艺路线

工艺路线说明产品经过哪些工序生产。

| 对象 | 说明 |
| --- | --- |
| 工序 | 生产步骤，如切割、焊接、装配、检验 |
| 工作中心 | 工序执行地点 |
| 资源 | 人员、设备、模具 |
| 标准工时 | 计划排产和成本基础 |
| 检验点 | 是否需要质检 |

工艺路线影响生产排程、工序汇报、计件工资和成本分配。

## 三、计划管理

### 需求来源

| 来源 | 说明 |
| --- | --- |
| 销售订单 | 以客户订单驱动生产 |
| 预测单 | 根据市场预测备货 |
| 安全库存 | 库存低于安全线补货 |
| 生产订单下级需求 | 半成品和材料需求 |
| 手工计划 | 计划员人工补充 |

### MRP 运行前检查

- 物料是否有采购、生产或委外属性。
- BOM 是否有效。
- 库存数量是否准确。
- 销售订单、预测单是否审核。
- 安全库存、最小批量、提前期是否维护。
- 在途采购、在制生产是否纳入计算。
- 仓库、组织、计划范围是否正确。

### MRP 输出

MRP 通常输出计划订单，计划员再投放为采购申请、采购订单、生产订单或委外订单。

| 输出 | 适用物料 |
| --- | --- |
| 采购建议 | 外购物料 |
| 生产建议 | 自制件 |
| 委外建议 | 委外件 |
| 调拨建议 | 其他仓库或组织有库存 |

## 四、生产订单执行

### 标准流程

```text
计划订单
  -> 生产订单
  -> 生产订单审核
  -> 备料
  -> 领料或倒冲
  -> 工序汇报
  -> 完工入库
  -> 关闭生产订单
```

### 生产订单检查

- 成品物料是否正确。
- 数量和单位是否正确。
- 计划开工、完工日期是否合理。
- BOM 版本是否正确。
- 工艺路线是否正确。
- 生产组织和生产车间是否正确。
- 是否允许超领、欠料开工、超产入库。

### 领料

领料方式：

| 方式 | 说明 | 风险 |
| --- | --- | --- |
| 按单领料 | 按生产订单领料 | 最清晰，但操作多 |
| 配套领料 | 按套料一次领 | 缺料要处理 |
| 倒冲领料 | 完工时自动扣料 | 依赖 BOM 准确 |
| 补料 | 生产损耗或异常补领 | 要审批 |
| 退料 | 多领或不良退回 | 要及时处理 |

领料检查：

- 材料是否按 BOM。
- 仓库、批号、库位是否正确。
- 是否超领。
- 替代料是否审批。
- 领料后库存是否正确减少。

### 工序汇报

如果企业启用工序管理，要记录每道工序的开工、完工、合格、不良、返工。

检查点：

- 工序顺序是否正确。
- 合格数、不良数是否合理。
- 是否需要质检。
- 工时是否记录。
- 是否影响计件工资或成本。

### 完工入库

完工入库是产成品或半成品进入库存的动作。

操作步骤：

1. 选择生产订单。
2. 录入完工数量。
3. 选择入库仓库和库位。
4. 录入批号或序列号。
5. 处理不良品或返工品。
6. 审核完工入库单。
7. 查询库存增加。

## 五、委外管理

委外是把某些工序或物料交给供应商加工。

```text
委外订单
  -> 委外发料
  -> 委外加工
  -> 委外入库
  -> 委外应付
  -> 委外成本核算
```

委外检查点：

- 委外供应商是否正确。
- 发出材料是否可追溯。
- 供应商库存是否管理。
- 委外入库数量是否等于加工完成数量。
- 委外加工费是否进入应付。
- 材料成本和加工费是否进入产品成本。

## 六、制造成本

### 成本构成

| 成本 | 来源 |
| --- | --- |
| 直接材料 | 生产领料或倒冲 |
| 直接人工 | 工时、工资分配 |
| 制造费用 | 水电、折旧、车间费用 |
| 委外加工费 | 委外应付 |
| 废品损失 | 不良品、报废 |

### 成本核算顺序

```text
材料出库核算
  -> 人工和制造费用归集
  -> 委外加工费确认
  -> 在产品分配
  -> 完工成本计算
  -> 成本凭证生成
```

成本异常常见原因：

- BOM 用量错误。
- 领料不完整或超领。
- 完工数量错误。
- 费用归集部门错误。
- 成本对象不清。
- 生产订单未关闭。

## 七、制造与财务对账

| 对账内容 | 目标 |
| --- | --- |
| 生产领料对库存 | 材料库存减少正确 |
| 完工入库对库存 | 成品库存增加正确 |
| 生产订单成本 | 材料、人工、费用归集完整 |
| 存货核算对总账 | 存货和成本科目一致 |
| 委外加工对供应商 | 委外应付和委外入库一致 |

## 八、常见问题

### MRP 计划数量异常

排查：

1. 查需求来源是否重复。
2. 查库存是否准确。
3. 查 BOM 是否有效。
4. 查提前期和批量规则。
5. 查在途采购和在制生产是否纳入。

### 生产订单缺料

排查：

1. 查生产订单用料清单。
2. 查库存组织、仓库、批号。
3. 查采购在途和调拨在途。
4. 查替代料。
5. 查是否允许欠料开工。

### 完工成本异常

排查：

1. 查领料是否完整。
2. 查是否有超领或补料。
3. 查人工和制造费用是否归集。
4. 查完工数量是否正确。
5. 查成本计算是否完成。

## 九、制造上线验收清单

- [ ] 物料生产属性正确。
- [ ] BOM 生效并经过工程确认。
- [ ] 工艺路线和工作中心确认。
- [ ] MRP 能输出合理计划订单。
- [ ] 生产订单能投放、审核、领料、完工。
- [ ] 委外流程能跑通。
- [ ] 生产领料和完工入库能影响库存。
- [ ] 成本核算能计算完工成本。
- [ ] 制造相关凭证能生成并过账。
- [ ] 生产、库存、成本、总账能对账。
