工程数据是生产制造的地基。BOM、工艺路线、工作中心、工厂日历、物料生产属性如果不准确，后续 MRP、生产订单、领料、完工、成本核算都会错。

## 一、工程数据大图

```text
物料生产属性
  -> 产品 BOM
  -> 工艺路线
  -> 工作中心和资源
  -> 工厂日历
  -> 替代料和损耗
  -> MRP 和生产订单
```

| 工程数据 | 作用 | 下游影响 |
| --- | --- | --- |
| 物料 | 决定采购、自制、委外、库存和成本属性 | 计划、采购、生产、库存 |
| BOM | 决定成品由哪些材料组成 | MRP 展开、领料、成本 |
| 工艺路线 | 决定生产经过哪些工序 | 排产、工序汇报、人工制造费用 |
| 工作中心 | 决定生产资源和能力 | 产能计划、成本中心 |
| 工厂日历 | 决定工作日和休息日 | 计划日期、开完工日期 |
| 替代料 | 缺料时替换材料 | 计划和领料 |

## 二、物料生产属性

物料不仅是库存资料，也是计划和生产的基础。

| 属性 | 说明 | 示例 |
| --- | --- | --- |
| 自制 | 企业内部生产 | 成品、半成品 |
| 外购 | 从供应商采购 | 原材料、辅料 |
| 委外 | 委托供应商加工 | 表面处理件、外协半成品 |
| 虚拟件 | 不实际库存，用于 BOM 分组展开 | 套件、组件包 |
| 批号管理 | 批次追踪 | 食品、药品、电子料 |
| 序列号管理 | 单件追踪 | 设备、整机 |

物料检查：

- 基本单位、库存单位、采购单位、生产单位是否一致或有正确换算。
- 物料是否允许生产或委外。
- 是否启用批号、保质期、序列号。
- 是否维护提前期、最小批量、固定批量。
- 是否维护默认仓库、发料仓库、完工入库仓库。
- 是否维护计划策略。

## 三、BOM 是什么

BOM 是产品结构，用来表达一个成品需要哪些原材料、零部件、半成品，以及它们之间的层级、用量和损耗。金蝶官方产品手册也把产品 BOM 定义为生产制造系统的基础数据，用来表达制成品所需原材料、零部件或半成品的组成关系。

```text
成品 A
  -> 半成品 B  1 件
      -> 原材料 B1  2 件
      -> 原材料 B2  3 件
  -> 原材料 C  4 件
  -> 包材 D    1 套
```

## 四、BOM 关键字段

| 字段 | 说明 | 风险 |
| --- | --- | --- |
| 父项物料 | 要生产的成品或半成品 | 选错会导致计划和成本全错 |
| BOM 版本 | 不同产品结构版本 | 版本未生效会用错料 |
| 子项物料 | 需要消耗的材料 | 漏料会缺料，错料会错成本 |
| 用量 | 单位成品耗用数量 | 用量错会导致领料和成本错 |
| 损耗率 | 正常生产损耗 | 过高会虚增需求和成本 |
| 生效日期 | BOM 可用期间 | 旧 BOM 未失效会混用 |
| 发料方式 | 按单领料、倒冲、调拨等 | 影响仓库和车间操作 |
| 替代料 | 缺料替代 | 需要审批和成本确认 |
| 工序 | 材料在哪道工序使用 | 工序领料和工序成本 |
| 货主 | 物料所有权 | 受托加工或客户料常用 |

## 五、BOM 建立步骤

1. 确认成品或半成品物料已建立。
2. 确认所有子项物料已建立。
3. 确认计量单位和换算率。
4. 新建 BOM。
5. 录入父项物料。
6. 录入子项物料、用量、损耗率、发料方式。
7. 如果启用工艺，维护子项使用工序。
8. 如果有替代料，维护替代关系。
9. 保存、提交、审核。
10. 做 BOM 展开检查。
11. 用一张生产订单测试用料清单。

## 六、BOM 版本管理

版本管理适合产品结构经常变化的企业。

版本变更要记录：

- 变更原因。
- 变更前后材料。
- 生效日期。
- 影响在制生产订单。
- 影响库存材料。
- 影响成本。
- 是否需要工程、生产、采购、财务共同确认。

常见规则：

- 新订单用新版本。
- 已下达生产订单是否同步更新，要单独评估。
- 已领料生产订单不要随意改 BOM。
- BOM 变更要通知采购和仓库。

## 七、替代料

替代料用于材料缺货时用其他物料替代。

替代料设计要确认：

- 替代比例。
- 是否完全等效。
- 是否影响质量。
- 是否影响成本。
- 是否需要审批。
- 是否允许 MRP 自动考虑替代料。

风险：

- 替代料成本差异影响毛利。
- 替代料没有库存导致计划误判。
- 替代料未同步给车间导致领错料。

## 八、工艺路线

工艺路线描述产品生产顺序。

```text
下料
  -> 加工
  -> 焊接
  -> 表面处理
  -> 装配
  -> 检验
  -> 包装
```

关键字段：

| 字段 | 说明 |
| --- | --- |
| 工序号 | 工序顺序 |
| 工序名称 | 生产步骤 |
| 工作中心 | 执行地点或资源 |
| 标准工时 | 计划和成本基础 |
| 是否检验 | 是否需要质检 |
| 是否委外 | 是否外协加工 |
| 是否报工 | 是否需要工序汇报 |

工艺路线影响：

- 生产订单排程。
- 工序汇报。
- 工序领料。
- 人工成本。
- 制造费用分配。

## 九、工作中心和工厂日历

### 工作中心

工作中心可以是车间、产线、设备组或工序资源。

需要维护：

- 所属生产组织。
- 所属部门。
- 产能。
- 班次。
- 成本中心。
- 是否瓶颈资源。

### 工厂日历

工厂日历决定哪些日期可以排产。

要维护：

- 工作日。
- 休息日。
- 节假日。
- 班次。
- 加班。

如果工厂日历错，MRP 建议开工完工日期会错。

## 十、BOM 到生产订单

BOM 审核后，会被生产订单引用并生成生产用料清单。生产用料清单决定车间实际要领哪些料。

```text
产品 BOM
  -> 生产订单
  -> 生产用料清单
  -> 领料单/倒冲领料
  -> 材料成本
```

检查点：

- 生产订单使用的 BOM 版本是否正确。
- 用料清单是否按 BOM 展开。
- 替代料是否正确。
- 损耗是否带入。
- 工序用料是否正确。

## 十一、BOM 到成本

BOM 是产品材料成本的重要基础。物料清单成本查询可按 BOM 层级汇总采购料单价金额，用于估算产成品、半成品材料成本。

成本关注：

- 材料用量。
- 材料单价。
- 损耗率。
- 半成品层级。
- 替代料成本。
- 联副产品分摊。

如果 BOM 用量错，生产领料和成本都会错。

## 十二、常见问题

### MRP 跑不出需求

可能原因：

- BOM 未审核。
- BOM 未生效。
- 子项物料不参与 MRP。
- 物料计划属性错误。
- 需求来源未审核。

### 生产订单用料不对

排查：

1. 查生产订单的 BOM 版本。
2. 查 BOM 是否变更。
3. 查用料清单是否手工修改。
4. 查替代料。
5. 查损耗率。

### 成本异常偏高

可能原因：

- BOM 用量偏高。
- 损耗率过高。
- 替代料成本高。
- 半成品成本异常。
- 生产超领。

## 十三、工程数据验收清单

- [ ] 自制、外购、委外物料属性正确。
- [ ] 成品和半成品 BOM 建立并审核。
- [ ] BOM 版本、生效日期正确。
- [ ] 子项物料、用量、损耗率正确。
- [ ] 替代料规则确认。
- [ ] 工艺路线和工作中心确认。
- [ ] 工厂日历确认。
- [ ] BOM 能正确生成生产用料清单。
- [ ] BOM 成本查询结果合理。
