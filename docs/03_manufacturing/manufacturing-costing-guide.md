制造成本核算的目标，是把生产过程中的材料、人工、制造费用、委外加工费等归集到产品成本，并区分完工产品和在产品。成本核算不是财务月末最后点一下，它依赖 BOM、领料、完工、费用归集和生产订单状态。

## 一、制造成本大图

```text
生产订单
  -> 材料领用
  -> 人工归集
  -> 制造费用归集
  -> 委外加工费
  -> 在产品分配
  -> 完工成本
  -> 成本凭证
```

## 二、成本构成

| 成本项目 | 来源 | 常见凭证影响 |
| --- | --- | --- |
| 直接材料 | 生产领料、倒冲领料 | 借生产成本，贷原材料 |
| 直接人工 | 工资、工时、计件 | 借生产成本，贷应付职工薪酬 |
| 制造费用 | 折旧、水电、维修、车间费用 | 借制造费用，再分配到生产成本 |
| 委外加工费 | 委外应付 | 进入委外产品成本 |
| 废品损失 | 报废、不良、返工 | 视政策进成本或损失 |

## 三、启用产品成本核算

官方资料说明，产品成本核算启用会决定期初在产品数据的取数来源，通常取启用期间前一天已下达未结案的生产订单和委外订单。启用前还要求存货核算已结束初始化。

启用前检查：

- 存货核算已经初始化完成。
- 生产订单和委外订单期初状态清楚。
- 在产品数量和金额可解释。
- 成本中心、成本项目、费用项目已维护。
- 会计政策和核算组织确认。

## 四、成本对象

成本可以按不同对象归集：

| 对象 | 适用 |
| --- | --- |
| 生产订单 | 离散制造、按单生产 |
| 产品 | 批量生产 |
| 成本中心 | 车间、产线 |
| 工序 | 精细工序成本 |
| 项目 | 项目制造 |

金蝶项目里要先确定客户想按什么维度看成本，否则后面报表和分摊规则会混乱。

## 五、材料成本

材料成本来自生产领料或倒冲。

```text
借：生产成本-直接材料
贷：原材料
```

检查点：

- 领料单是否审核。
- 领料是否关联生产订单。
- 材料出库成本是否计算。
- 是否存在负库存。
- 补料、退料是否处理。
- 替代料成本是否合理。

## 六、人工和制造费用

### 人工费用

人工费用来源：

- 工时。
- 计件。
- 工资分配。
- 手工录入。

检查：

- 工时是否准确。
- 人员或班组是否归属正确。
- 成本中心是否正确。
- 分配规则是否合理。

### 制造费用

制造费用包括：

- 车间折旧。
- 水电费。
- 维修费。
- 辅料消耗。
- 车间管理人员工资。

制造费用通常先归集到制造费用科目，再按工时、产量、机器小时或其他规则分配到产品成本。

## 七、在产品和完工成本

月末生产订单可能有未完工部分，这就是在产品。

成本分配要回答：

- 本期投入多少。
- 本期完工多少。
- 期末在产品多少。
- 材料是否一次投入。
- 人工和费用按什么比例分配。

常见分配方式：

| 方式 | 适用 |
| --- | --- |
| 不计算在产品 | 生产周期短 |
| 约当产量 | 在产品较多且进度可估 |
| 定额成本 | 成本稳定 |
| 按完工比例 | 工序进度明确 |

## 八、成本计算

官方成本计算资料提示：成本计算执行后要看合法性检查报告。如果检查结果为错误、异常、不通过，说明成本计算未全部完成，不应该直接分析数据准确性，要先处理问题并重新计算。

### 操作步骤

1. 清理生产和库存单据。
2. 完成材料出库核算。
3. 归集人工和制造费用。
4. 确认委外加工费。
5. 执行产品成本计算。
6. 查看合法性检查报告。
7. 处理错误和异常。
8. 查看成本计算单。
9. 生成成本凭证。
10. 与总账对账。

### 合法性检查常见问题

- 生产订单未完工但状态异常。
- 领料没有成本。
- 完工入库没有成本。
- 成本中心缺失。
- 费用项目缺失。
- 在产品数量异常。
- 负库存。

## 九、成本计算单

成本计算单用于查看工单或成本对象的成本结果。官方字段说明中，成本计算单通常包含核算体系、核算组织、会计政策、期间、业务类型、成本中心、工单编号、产品编码、成本项目、费用项目等字段。

看成本计算单要关注：

- 本期投入。
- 本期完工。
- 期末在产。
- 材料成本。
- 人工成本。
- 制造费用。
- 委外加工费。
- 单位成本。

## 十、成本凭证

常见分录：

### 生产领料

```text
借：生产成本-直接材料
贷：原材料
```

### 归集制造费用

```text
借：制造费用
贷：累计折旧/应付职工薪酬/银行存款等
```

### 制造费用分配

```text
借：生产成本-制造费用
贷：制造费用
```

### 完工入库

```text
借：库存商品
贷：生产成本
```

### 销售结转成本

```text
借：主营业务成本
贷：库存商品
```

## 十一、制造成本对账

| 对账 | 目标 |
| --- | --- |
| 生产领料对原材料 | 原材料减少和生产成本增加一致 |
| 完工入库对库存商品 | 库存商品增加等于完工成本 |
| 生产成本余额 | 未完工订单形成在产品 |
| 制造费用 | 月末应分配完或有合理余额 |
| 成本计算单对总账 | 成本结果和凭证一致 |

## 十二、常见问题

### 完工成本为 0

可能原因：

- 未领料。
- 领料未核算成本。
- 存货核算未完成。
- 成本计算未成功。
- 完工入库未参与成本。

### 单位成本异常高

排查：

1. 生产数量是否偏低。
2. 材料是否超领。
3. 补料是否异常。
4. 制造费用分配是否过高。
5. 是否把其他订单费用分配进来。

### 生产成本余额异常

排查：

1. 未完工订单是否真实存在。
2. 完工入库是否漏做。
3. 退料是否漏做。
4. 完工成本是否结转。
5. 生产订单是否结案。

## 十三、制造成本月结清单

- [ ] 生产订单状态检查完成。
- [ ] 生产领料、补料、退料全部审核。
- [ ] 完工入库全部审核。
- [ ] 材料出库成本核算完成。
- [ ] 人工和制造费用归集完成。
- [ ] 委外加工费确认完成。
- [ ] 产品成本计算完成。
- [ ] 合法性检查报告无错误。
- [ ] 成本凭证生成并过账。
- [ ] 成本报表和总账对账完成。
