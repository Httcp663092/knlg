委外管理用于企业把部分加工任务交给外部供应商完成。委外和普通采购不同：通常企业提供材料，供应商负责加工，完工品产权属于委托企业。金蝶官方委外管理流程也强调先设基础数据和工程数据，再设置委外参数，然后执行委外订单、委外领料等日常业务。

## 一、委外业务流程

```text
需求来源
  -> 委外申请/委外订单
  -> 委外用料清单
  -> 委外发料/领料
  -> 委外加工
  -> 委外入库
  -> 委外应付
  -> 委外入库核算
  -> 总账凭证
```

## 二、委外前置资料

| 资料 | 作用 |
| --- | --- |
| 委外物料 | 标识哪些物料委外加工 |
| BOM | 决定委外需要发哪些材料 |
| 委外供应商 | 加工方 |
| 委外车间 | 委外生产管理对象 |
| 工厂日历 | 计算交期 |
| 发料仓库 | 委外材料出库 |
| 委外参数 | 控制领料、入库、核算方式 |
| 价格和加工费 | 形成委外应付 |

## 三、委外订单

委外订单是交给供应商执行的加工任务。

### 操作步骤

1. 选择委外组织。
2. 选择委外供应商。
3. 选择委外物料。
4. 录入委外数量。
5. 确认计划开工和完工日期。
6. 确认 BOM 和用料。
7. 保存、提交、审核。
8. 生成委外用料清单。

### 检查点

- 委外供应商是否可用。
- 委外物料属性是否正确。
- BOM 是否有效。
- 发料仓库是否正确。
- 加工单价是否明确。
- 交货日期是否合理。

## 四、委外用料清单

委外用料清单决定发给供应商哪些材料。

检查：

- 子项物料是否完整。
- 用量是否按 BOM。
- 损耗是否合理。
- 发料方式是否正确。
- 供应商库存是否管理。
- 替代料是否允许。

如果委外用料清单不准确，会导致供应商缺料、发料不准、委外成本不准。

## 五、委外发料和领料

委外发料把企业材料发给供应商。

### 操作步骤

1. 选择委外订单。
2. 选择委外用料清单。
3. 生成委外领料单或发料单。
4. 选择发料仓库、批号、库位。
5. 审核发料单。
6. 查询企业库存减少或供应商库存增加。

### 检查点

- 对应委外用料清单是否审核。
- 发料数量是否超用料。
- 批号是否可追溯。
- 是否管理供应商仓库存。
- 多发、少发、补料、退料是否受控。

## 六、委外补料和退料

### 补料

适用：

- 加工损耗超预期。
- 材料报废。
- BOM 漏料。
- 客户临时变更。

补料要审批，否则委外成本会失控。

### 退料

适用：

- 发多了。
- 订单取消。
- 材料不合格。
- 完工后剩料退回。

退料要关联委外订单和原发料，保证供应商库存和企业库存一致。

## 七、委外入库

委外入库是供应商加工完成后，企业收回成品或半成品。

操作步骤：

1. 根据委外订单生成委外入库单。
2. 录入入库数量。
3. 选择仓库、库位、批号。
4. 处理质检。
5. 审核入库单。
6. 查询库存增加。

检查点：

- 入库数量是否超过委外订单。
- 是否有质检。
- 批号是否正确。
- 未完工数量是否合理。
- 是否需要委外应付。

## 八、委外应付和加工费

委外加工费一般通过应付确认。

```text
委外入库
  -> 委外应付
  -> 付款
  -> 核销
```

应付检查：

- 供应商。
- 加工数量。
- 加工单价。
- 税率。
- 价税合计。
- 应付凭证。

## 九、委外入库核算

委外入库核算用于计算委外加工产品成本。官方资料说明，委外加工产品成本通常包括两部分：耗用委外发出物料的材料成本，以及支付给加工厂商的加工费和附加费用。

成本构成：

```text
委外成品成本
  = 发出材料成本
  + 委外加工费
  + 其他附加费用
```

检查点：

- 发料材料成本是否完整。
- 委外加工费是否确认。
- 运费、装卸费是否分摊。
- 委外入库核算是否完成。
- 委外成本凭证是否生成。

## 十、委外和财务对账

| 对账 | 目标 |
| --- | --- |
| 委外发料对供应商库存 | 发出材料可追溯 |
| 委外订单对委外入库 | 加工任务完成情况清楚 |
| 委外应付对供应商 | 加工费应付完整 |
| 委外入库核算对存货 | 委外产品成本完整 |
| 委外凭证对总账 | 存货、应付、成本一致 |

## 十一、常见问题

### 委外领料失败

可能原因：

- 委外用料清单未审核。
- 库存不足。
- 批号或库位缺失。
- 发料仓库错误。
- 委外订单未审核。

### 委外入库成本不完整

排查：

1. 委外发料是否完整。
2. 发出材料成本是否核算。
3. 委外应付是否确认。
4. 加工费是否正确。
5. 委外入库核算是否完成。

### 供应商处材料对不上

排查：

1. 查委外发料。
2. 查委外退料。
3. 查委外补料。
4. 查委外入库耗用。
5. 和供应商对账。

## 十二、委外上线验收清单

- [ ] 委外物料和委外供应商配置正确。
- [ ] 委外 BOM 能生成用料清单。
- [ ] 委外订单能创建、审核。
- [ ] 委外领料、补料、退料跑通。
- [ ] 委外入库跑通。
- [ ] 委外应付和付款跑通。
- [ ] 委外入库核算能计算成本。
- [ ] 委外相关凭证能生成并过账。
- [ ] 供应商材料对账可用。
