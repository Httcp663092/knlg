供应链和库存数据排查要同时看单据状态、上下游关系、库存数量、核算金额和财务凭证。采购、销售、库存、生产、存货核算任何一个环节口径不一致，都会造成未结数量、库存数量、成本金额和总账余额差异。

## 一、供应链数据排查主线

```text
业务订单
  -> 出入库单据
  -> 即时库存
  -> 库存明细账
  -> 存货核算
  -> 凭证
  -> 总账
```

排查不要跳层。例如库存金额异常，先看库存业务和存货核算，再看总账；不要直接拿即时库存数量去对总账金额。

## 二、采购数据排查

### 采购到付款链路

```text
采购申请
  -> 采购订单
  -> 收料通知
  -> 采购入库
  -> 退料
  -> 应付单或采购发票
  -> 付款单
  -> 核销
```

### 常见问题

| 现象 | 排查方向 |
| --- | --- |
| 采购订单未结数量不对 | 入库反写、关闭状态、变更记录 |
| 入库后库存没增加 | 入库单状态、仓库、库存状态、批号 |
| 入库后应付没生成 | 单据类型、业务流程、应付生成规则 |
| 暂估金额不对 | 入库单价、税率、币别、价税合计 |
| 付款无法核销 | 供应商、币别、组织、应付状态不一致 |

### 查询模板

```sql
-- 查询采购订单执行情况的结构模板
select
  采购订单号,
  供应商,
  物料编码,
  订单数量,
  累计入库数量,
  未入库数量,
  关闭状态
from 采购订单执行视图
where 采购组织 = '目标组织'
  and 订单日期 between '开始日期' and '结束日期';
```

验证方法：

- 用订单号回前台联查入库单。
- 比较订单分录数量和入库分录数量。
- 检查关闭状态是否人为关闭。

## 三、销售数据排查

### 销售到收款链路

```text
销售订单
  -> 发货通知
  -> 销售出库
  -> 销售退货
  -> 应收单或销售发票
  -> 收款单
  -> 核销
```

### 常见问题

| 现象 | 排查方向 |
| --- | --- |
| 销售订单可发数量不对 | 库存、锁库、关闭、已发数量 |
| 出库后应收没生成 | 单据类型、结算方式、生成规则 |
| 销售毛利异常 | 出库成本、销售收入、退货冲回 |
| 客户未收款余额不对 | 应收、收款、核销、总账 |
| 信用控制未生效 | 客户信用资料、单据状态、参数 |

### 查询模板

```sql
-- 查询销售订单出库和应收情况的结构模板
select
  销售订单号,
  客户,
  物料编码,
  订单数量,
  出库数量,
  应收金额,
  未收金额
from 销售订单执行视图
where 销售组织 = '目标组织'
  and 订单日期 between '开始日期' and '结束日期';
```

## 四、库存数量排查

### 库存数量层级

| 层级 | 用途 |
| --- | --- |
| 即时库存 | 当前可用、锁定、在库数量 |
| 库存明细账 | 某期间收发存流水 |
| 库存余额 | 按期间、仓库、物料汇总 |
| 批号库存 | 按批号追踪 |
| 库位库存 | 按库位追踪 |

### 库存异常分类

| 异常 | 可能原因 |
| --- | --- |
| 负库存 | 出库早于入库、允许负库存、补单 |
| 有账无物 | 盘点未调整、历史单据错误 |
| 有物无账 | 漏入库、未审核单据、线下收货 |
| 批号不一致 | 批号录入不规范、拆批合批 |
| 库位不一致 | 移库未做、库位必录控制不足 |

### 库存排查模板

```sql
-- 查询某物料库存分布的结构模板
select
  物料编码,
  物料名称,
  仓库,
  库位,
  批号,
  库存状态,
  数量
from 即时库存视图
where 物料编码 = '目标物料'
  and 组织 = '目标库存组织';
```

验证：

- 前台即时库存查询。
- 库存明细账收发存。
- 最近出入库单据。
- 盘点记录。

## 五、库存金额和存货核算排查

数量正确不代表金额正确。存货金额依赖存货核算。

### 核算链路

```text
库存单据
  -> 入库成本或暂估成本
  -> 出库成本计算
  -> 存货核算报表
  -> 存货凭证
  -> 总账
```

### 常见问题

| 现象 | 可能原因 |
| --- | --- |
| 出库成本为 0 | 入库无价、成本计算未完成 |
| 成本单价异常高 | 负库存、调整金额、计价范围错误 |
| 存货和总账不一致 | 凭证未生成、未过账或手工凭证 |
| 暂估差异未处理 | 到票核算或暂估冲回未完成 |
| 退货成本异常 | 原出库成本未正确关联 |

### 检查步骤

1. 查库存单据是否审核。
2. 查入库单价是否存在。
3. 查计价方法和计价范围。
4. 查出库成本计算结果。
5. 查存货核算报表。
6. 查凭证生成和过账。
7. 查总账存货科目。

## 六、生产库存排查

生产相关库存包括：

- 生产领料。
- 补料。
- 退料。
- 完工入库。
- 委外发料。
- 委外入库。
- 在制品。

### 常见问题

| 现象 | 排查方向 |
| --- | --- |
| 生产订单缺料 | BOM、可用库存、锁库、需求日期 |
| 领料数量超标准 | 超领参数、损耗、替代料、手工领料 |
| 完工入库后成本不对 | 领料成本、费用分配、完工数量 |
| 委外材料对不上 | 委外发料、委外入库、未结数量 |

## 七、上下游断链排查

### 表现

- 前台联查不到上游或下游。
- 订单执行数量不更新。
- 已关闭但仍显示未结。
- 下游单据删除后上游状态异常。

### 排查顺序

1. 查源单是否存在并审核。
2. 查下游单据是否存在。
3. 查关联关系。
4. 查累计执行数量。
5. 查关闭状态。
6. 查转换规则是否修改过。

## 八、实际案例

### 案例：销售出库后库存减少，但总账存货没减少

现象：

- 销售出库单已审核。
- 即时库存数量已减少。
- 总账存货科目没有减少。

排查：

1. 查销售出库单状态，确认已审核。
2. 查存货核算，发现该期间未执行出库成本核算。
3. 执行成本计算后，存货核算报表出现出库成本。
4. 查凭证生成，发现出库成本凭证未生成。
5. 生成凭证并过账后，总账存货减少。

结论：

- 库存数量减少属于库存模块结果。
- 总账金额减少必须经过存货核算和凭证过账。

## 九、供应链 SQL 排查清单

- [ ] 业务组织、库存组织、核算组织已确认。
- [ ] 单据日期和会计期间已确认。
- [ ] 单据状态是否审核已确认。
- [ ] 上下游关联已检查。
- [ ] 即时库存和库存明细账已对比。
- [ ] 批号、库位、库存状态已检查。
- [ ] 存货核算是否完成已检查。
- [ ] 凭证是否生成和过账已检查。
- [ ] 差异已按数量、金额、凭证分类。
