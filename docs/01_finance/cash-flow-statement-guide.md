现金流量表反映企业在一定期间内现金和现金等价物的流入、流出情况。利润表显示赚不赚钱，现金流量表显示钱有没有真正进来、流出去哪里。金蝶里现金流量表通常依赖现金流项目、凭证指定、收付款单据和报表公式，配置不清会导致现金流量表和银行存款变化对不上。

## 一、现金流量表的三大类

| 分类 | 说明 | 示例 |
| --- | --- | --- |
| 经营活动现金流 | 日常经营产生的现金流 | 销售收款、采购付款、支付工资、缴税 |
| 投资活动现金流 | 长期资产和投资相关现金流 | 购买固定资产、处置设备、投资收回 |
| 筹资活动现金流 | 融资和资本相关现金流 | 借款、还款、分红、吸收投资 |

## 二、现金流量表和利润表的区别

利润表按权责发生制：

```text
收入确认不一定收到现金
费用确认不一定支付现金
```

现金流量表按现金收付：

```text
现金流入流出才影响现金流量表
```

例子：

- 销售 100 万，客户未付款：利润表有收入，现金流量表没有销售收款。
- 采购固定资产付款 50 万：现金流量表有投资流出，利润表不一定当期确认费用。

## 三、金蝶中的现金流逻辑

现金流量表通常依赖：

- 现金类科目。
- 银行类科目。
- 现金流项目。
- 凭证现金流量指定。
- 收付款单据。
- 报表公式。

```text
收付款单据 / 凭证
  -> 现金银行科目
  -> 指定现金流项目
  -> 现金流量表取数
```

## 四、现金流项目

现金流项目是现金流量表的分类标签。

常见项目：

### 经营活动

- 销售商品、提供劳务收到的现金。
- 收到的税费返还。
- 收到其他与经营活动有关的现金。
- 购买商品、接受劳务支付的现金。
- 支付给职工以及为职工支付的现金。
- 支付的各项税费。
- 支付其他与经营活动有关的现金。

### 投资活动

- 收回投资收到的现金。
- 取得投资收益收到的现金。
- 处置固定资产、无形资产收到的现金。
- 购建固定资产、无形资产支付的现金。
- 投资支付的现金。

### 筹资活动

- 吸收投资收到的现金。
- 取得借款收到的现金。
- 偿还债务支付的现金。
- 分配股利、利润或偿付利息支付的现金。

## 五、凭证指定现金流

如果凭证涉及现金银行科目，通常需要指定现金流项目。

示例：收到客户货款。

```text
借：银行存款 113,000
  贷：应收账款 / 客户A 113,000
```

现金流项目：

```text
销售商品、提供劳务收到的现金
```

示例：支付供应商货款。

```text
借：应付账款 / 供应商B 56,500
  贷：银行存款 56,500
```

现金流项目：

```text
购买商品、接受劳务支付的现金
```

## 六、自动指定和手工指定

金蝶项目中常见两种方式：

| 方式 | 说明 | 风险 |
| --- | --- | --- |
| 自动指定 | 根据凭证模板、科目或业务单据自动带出 | 规则不全会漏指定 |
| 手工指定 | 会计在凭证上手工选择现金流项目 | 容易选错或漏选 |

建议：

- 高频业务用自动指定。
- 复杂调整凭证允许手工指定。
- 月末检查未指定现金流凭证。

## 七、现金流项目判断方法

### 看业务实质

不要只看对方科目，要看现金流发生原因。

| 业务 | 现金流项目 |
| --- | --- |
| 客户回款 | 销售商品、提供劳务收到的现金 |
| 支付供应商货款 | 购买商品、接受劳务支付的现金 |
| 支付工资 | 支付给职工以及为职工支付的现金 |
| 缴纳增值税和所得税 | 支付的各项税费 |
| 购买设备 | 购建固定资产、无形资产支付的现金 |
| 收到银行借款 | 取得借款收到的现金 |
| 偿还银行本金 | 偿还债务支付的现金 |
| 支付借款利息 | 分配股利、利润或偿付利息支付的现金 |

### 同一笔付款可能拆分

例如支付供应商 113,000，其中 100,000 是货款，13,000 是税额。现金流项目一般仍属于购买商品、接受劳务支付的现金，不需要按税额拆分到缴税。

## 八、现金流量表检查

现金流量表底层平衡关系：

```text
期初现金及现金等价物余额
+ 本期现金净增加额
= 期末现金及现金等价物余额
```

检查顺序：

1. 银行存款、现金科目期初期末是否正确。
2. 本期现金银行凭证是否全部过账。
3. 是否有未指定现金流项目的凭证。
4. 现金流项目方向是否正确。
5. 内部银行转账是否被重复统计。
6. 外币现金流是否按正确汇率折算。

## 九、内部转账处理

银行账户之间转账：

```text
借：银行存款-A账户
  贷：银行存款-B账户
```

这类交易不会增加或减少企业总体现金，通常不应计入现金流量表的经营、投资或筹资流入流出。

常见错误：

- 一边指定为现金流入，另一边指定为现金流出，导致虚增现金流。
- 把内部账户调拨当成经营收款。

## 十、常见问题

| 问题 | 原因 | 处理 |
| --- | --- | --- |
| 现金流量表不平 | 现金科目范围、未指定项目、内部转账处理错误 | 查现金科目和未指定凭证 |
| 销售收款现金流少 | 收款凭证未指定或凭证未过账 | 查收款单和凭证 |
| 购买固定资产进经营流 | 现金流项目选错 | 改为投资活动项目 |
| 银行互转虚增现金流 | 内部转账未排除 | 设置不参与现金流或特殊项目 |
| 外币现金流异常 | 汇率或折算口径错误 | 检查币别和汇率 |

## 十一、实际案例：付款混合用途

业务：

- 企业向供应商付款 120,000。
- 其中 100,000 是采购货款。
- 20,000 是设备维修服务费。

处理方式：

如果凭证分录分清用途：

```text
借：应付账款-货款 100,000
借：管理费用-维修费 20,000
  贷：银行存款 120,000
```

现金流项目可拆分：

- 100,000：购买商品、接受劳务支付的现金。
- 20,000：支付其他与经营活动有关的现金。

如果系统无法自动拆分，需要手工指定或完善凭证模板。

## 十二、上线检查清单

- [ ] 现金和银行科目范围已确认。
- [ ] 现金流项目体系已确认。
- [ ] 收款、付款、费用、工资、税费、固定资产、借款等模板已测试。
- [ ] 内部转账不重复计入现金流。
- [ ] 未指定现金流凭证检查表已建立。
- [ ] 现金流量表和现金银行科目变动能核对。
- [ ] 外币现金流折算口径已确认。
- [ ] 月结前现金流检查责任人已明确。
