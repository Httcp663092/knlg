## 适用范围

适用于现金、银行、收付款流水、银行对账、资金余额、票据和网上银行相关场景。

## 核心目标

出纳与资金模块关注资金收付真实性、银行流水完整性、现金银行余额准确性，以及与总账现金银行科目的核对。

## 常见对象

| 对象 | 说明 |
| --- | --- |
| 收款单 | 客户回款或其他收入款项 |
| 付款单 | 供应商付款或其他支出款项 |
| 收付款退款单 | 收款或付款的反向处理 |
| 现金存取单 | 现金与银行之间的存取 |
| 银行转账单 | 银行账户之间的资金转移 |
| 银行日记账 | 银行账户流水记录 |
| 银行对账单 | 银行侧流水，用于对账 |

## 日常流程

1. 根据业务生成或录入收付款单。
2. 出纳审核资金信息、账户、金额、结算方式。
3. 需要时进行银行付款或银企直连处理。
4. 导入或维护银行对账单。
5. 执行银行对账。
6. 与总账现金银行科目核对。

## 银行对账实务

银行对账要同时看两边：

- 企业账：金蝶里的银行日记账、收付款单、付款单、银行转账单等。
- 银行账：银行对账单或银企直连回传流水。

### 对账步骤

1. 确认银行账号、币别和期间。
2. 导入或获取银行对账单。
3. 检查是否有重复导入流水。
4. 按金额、日期、摘要、对方户名进行自动或手工匹配。
5. 形成已达账和未达账。
6. 对未达账写说明。
7. 月末把银行余额、日记账余额和总账银行科目余额核对。

### 未达账项类型

| 类型 | 说明 | 处理 |
| --- | --- | --- |
| 银行已收，企业未收 | 银行流水已有，金蝶收款单未录 | 补录收款单或查接口失败 |
| 银行已付，企业未付 | 银行流水已有，金蝶付款单未录 | 补录付款单或查付款流程 |
| 企业已收，银行未收 | 金蝶已录收款，银行暂未到账 | 跟踪到账情况 |
| 企业已付，银行未付 | 金蝶已录付款，银行暂未扣款 | 跟踪银行处理 |

## 资金凭证模板设计要点

| 单据 | 借方 | 贷方 | 关键维度 |
| --- | --- | --- | --- |
| 收款单 | 银行存款 | 应收账款/其他应收款 | 银行账号、客户 |
| 付款单 | 应付账款/其他应付款 | 银行存款 | 供应商、银行账号 |
| 银行转账单 | 转入银行存款 | 转出银行存款 | 转入账号、转出账号 |
| 现金存取单 | 库存现金/银行存款 | 银行存款/库存现金 | 现金账户、银行账号 |

## 出纳月末检查表

- [ ] 收款单、付款单全部审核。
- [ ] 收款退款单、付款退款单全部审核。
- [ ] 银行转账单、现金存取单全部审核。
- [ ] 银行对账单导入完整。
- [ ] 银行对账完成。
- [ ] 未达账项有说明。
- [ ] 现金银行凭证全部生成。
- [ ] 凭证全部审核和过账。
- [ ] 出纳余额和总账资金科目一致。

## 月结关注点

- 是否存在未审核的收付款单、退款单、现金存取单、银行转账单。
- 银行日记账和银行对账单是否完成对账。
- 现金银行科目余额是否与出纳余额一致。
- 涉及现金银行科目的凭证是否完成出纳复核和过账。

## 常见问题

### 银行对账不平

检查银行账号、币别、日期、金额方向、手续费、未达账项和重复导入流水。

### 出纳余额和总账余额不一致

检查是否存在未生成凭证、未过账凭证、手工凭证、跨期间单据、账户科目映射错误。

### 收付款单无法生成凭证

检查智能会计平台规则、科目映射、核算维度、单据状态和期间状态。
