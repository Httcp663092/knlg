# 出纳与资金

## 适用范围

适用于现金、银行、收付款流水、银行对账、资金余额、票据和网上银行相关场景。

## 核心目标

出纳与资金模块关注资金收付真实性、银行流水完整性、现金银行余额准确性，以及与总账现金银行科目的核对。

## 常见对象

| 对象 | 说明 |
| --- | --- |
| 收款单 | 客户回款或其他收入款项 |
| 付款单 | 供应商付款或其他支出款项 |
| 收付款退款单 | 收款或付款的反向处理 |
| 现金存取单 | 现金与银行之间的存取 |
| 银行转账单 | 银行账户之间的资金转移 |
| 银行日记账 | 银行账户流水记录 |
| 银行对账单 | 银行侧流水，用于对账 |

## 日常流程

1. 根据业务生成或录入收付款单。
2. 出纳审核资金信息、账户、金额、结算方式。
3. 需要时进行银行付款或银企直连处理。
4. 导入或维护银行对账单。
5. 执行银行对账。
6. 与总账现金银行科目核对。

## 月结关注点

- 是否存在未审核的收付款单、退款单、现金存取单、银行转账单。
- 银行日记账和银行对账单是否完成对账。
- 现金银行科目余额是否与出纳余额一致。
- 涉及现金银行科目的凭证是否完成出纳复核和过账。

## 常见问题

### 银行对账不平

检查银行账号、币别、日期、金额方向、手续费、未达账项和重复导入流水。

### 出纳余额和总账余额不一致

检查是否存在未生成凭证、未过账凭证、手工凭证、跨期间单据、账户科目映射错误。

### 收付款单无法生成凭证

检查智能会计平台规则、科目映射、核算维度、单据状态和期间状态。
