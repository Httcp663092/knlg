这篇偏“每天怎么操作”。菜单名称会因金蝶产品版本、权限、客户配置和二开略有差异，实际以当前环境为准。记录时建议把客户环境里的真实菜单路径补到每个小节后面。

## 操作前先确认

每天做财务处理前，先固定四个口径：

| 口径 | 为什么重要 | 常见错误 |
| --- | --- | --- |
| 组织 | 决定业务单据和财务数据归属 | 把 A 公司业务录到 B 公司 |
| 账簿 | 决定凭证、科目余额和报表口径 | 查错账簿导致对账差异 |
| 会计期间 | 决定凭证期间和月结状态 | 业务日期跨期或期间已关闭 |
| 币别 | 决定金额和汇率 | 本位币、原币查询口径混用 |

建议每天开始前先看：

- 当前处理的是哪个核算组织。
- 当前账簿是否正确。
- 当前期间是否开放。
- 是否有待审核单据、未生成凭证单据、未过账凭证。

## 一、总账凭证日常操作

### 适用场景

- 业务单据自动生成凭证后，需要审核、过账。
- 财务需要录入少量手工调整凭证。
- 月末需要检查未审核、未过账凭证。

### 常见路径

```text
财务会计
  -> 总账
  -> 凭证管理
  -> 凭证录入 / 凭证查询 / 凭证审核 / 凭证过账
```

### 操作步骤：业务凭证审核过账

1. 进入凭证查询或凭证列表。
2. 固定查询条件：组织、账簿、期间、凭证状态。
3. 筛选“已生成未审核”的凭证。
4. 抽查凭证来源单据、科目、金额、辅助核算和摘要。
5. 确认无误后审核凭证。
6. 筛选“已审核未过账”的凭证。
7. 过账。
8. 查科目余额表，确认金额进入总账。

### 操作步骤：手工凭证录入

1. 确认这笔业务是否应该由子模块生成凭证。
2. 如果必须手工录入，先准备审批依据和附件。
3. 选择正确凭证字、凭证日期、账簿和期间。
4. 录入摘要，摘要要能说明业务原因。
5. 录入借方科目和贷方科目。
6. 补齐辅助核算，例如客户、供应商、部门、项目。
7. 上传或关联附件。
8. 保存、提交、审核、过账。
9. 在月结检查中单独复核手工凭证。

### 检查点

- 借贷是否平衡。
- 凭证日期是否属于正确期间。
- 科目是否和业务含义一致。
- 辅助核算是否完整。
- 来源单据是否能反查。
- 往来、资产、存货科目是否避免手工凭证。

### 常见异常

| 现象 | 可能原因 | 处理 |
| --- | --- | --- |
| 凭证不能审核 | 借贷不平、字段缺失、权限不足 | 查凭证完整性和权限 |
| 凭证不能过账 | 期间关闭、科目禁用、辅助核算缺失 | 打开期间或修正凭证 |
| 凭证查不到源单 | 手工凭证或关联丢失 | 回查生成日志和源单编号 |
| 科目余额没变化 | 凭证未过账或查询条件错 | 查凭证状态和账簿期间 |

## 二、应收日常操作：应收确认、收款、核销

### 适用场景

- 销售业务确认应收。
- 客户回款后录入收款。
- 收款和应收单核销。
- 查询客户余额、账龄和对账单。

### 常见路径

```text
财务会计
  -> 应收款管理
  -> 销售应收 / 其他应收 / 收款 / 核销 / 账表
```

### 操作步骤：确认应收

1. 从销售出库、销售发票或业务流程生成应收单。
2. 检查客户、结算客户、销售组织、结算组织。
3. 检查业务日期、到期日、币别、税率、价税合计。
4. 审核应收单。
5. 生成应收凭证。
6. 审核并过账凭证。
7. 查询客户应收明细或账龄。

### 操作步骤：录入收款并核销

1. 根据银行到账或收款通知录入收款单。
2. 检查客户、收款组织、结算方式、银行账号、金额和币别。
3. 审核收款单。
4. 生成收款凭证。
5. 进入核销功能，按客户、币别、组织筛选可核销单据。
6. 选择收款单和对应应收单。
7. 执行核销。
8. 查询应收账龄，确认该笔应收未收金额变为 0。

### 核销方式怎么选

| 核销方式 | 适合场景 | 注意点 |
| --- | --- | --- |
| 关联关系核销 | 收款由应收流程直接下推 | 最稳定，优先使用 |
| 匹配条件核销 | 同客户、同币别、同组织自动匹配 | 条件要谨慎，避免错配 |
| 手工核销 | 财务人工选择单据 | 要逐笔核对单据和金额 |
| 特殊核销 | 尾差、特殊往来清理 | 要审批，不能滥用 |
| 反核销 | 核销错误需要撤回 | 要确认对账和账龄影响 |

### 检查点

- 收款单是否已经审核。
- 收款凭证是否已经过账。
- 收款是否核销到正确应收单。
- 客户余额和账龄是否更新。
- 银行日记账是否和银行流水一致。

### 常见异常

| 现象 | 可能原因 | 处理 |
| --- | --- | --- |
| 找不到应收单 | 客户、组织、币别、状态不一致 | 固定条件后重新查询 |
| 核销失败 | 金额方向不匹配、已核销、期间关闭 | 查核销记录和单据状态 |
| 账龄仍显示逾期 | 收款未核销或核销错单 | 重新核销并复核账龄 |
| 收款凭证科目错 | 银行账号或凭证模板映射错 | 检查银行账号和模板 |

## 三、应付日常操作：应付确认、付款、核销

### 适用场景

- 采购业务确认应付。
- 费用或其他业务形成应付。
- 付款后核销应付。
- 查询供应商余额和账龄。

### 常见路径

```text
财务会计
  -> 应付款管理
  -> 采购应付 / 其他应付 / 付款 / 核销 / 账表
```

### 操作步骤：确认采购应付

1. 由采购入库、采购发票或业务流程生成应付单。
2. 检查供应商、结算供应商、采购组织、结算组织。
3. 检查业务日期、到期日、币别、税率、价税合计。
4. 审核应付单。
5. 生成应付凭证。
6. 审核并过账凭证。
7. 查询供应商应付明细或账龄。

### 操作步骤：付款并核销

1. 根据付款申请或付款计划生成付款单。
2. 检查供应商、付款组织、银行账号、结算方式、金额和币别。
3. 审核付款单。
4. 生成付款凭证。
5. 进入付款核销，按供应商、币别、组织筛选可核销单据。
6. 选择付款单和对应应付单。
7. 执行核销。
8. 查询应付账龄，确认该笔应付未付金额变为 0。

### 付款前控制

付款不是只看付款单金额。建议付款前检查：

- 供应商银行账号是否经过审批。
- 应付单是否已经审核。
- 发票或入库资料是否齐全。
- 是否已到付款条件。
- 是否存在重复付款风险。
- 是否有预付款、扣款、折让或尾差。

### 常见异常

| 现象 | 可能原因 | 处理 |
| --- | --- | --- |
| 应付单生成不了凭证 | 科目映射缺失、供应商维度缺失 | 检查凭证模板和供应商资料 |
| 付款单不能审核 | 权限、审批流、银行账号问题 | 查审批节点和账号资料 |
| 付款后账龄仍有余额 | 付款未核销或核销到其他单 | 查付款核销记录 |
| 应付和总账不一致 | 未过账、手工凭证、查询条件不同 | 固定条件做对账 |

## 四、出纳与银行日常操作

### 适用场景

- 记录现金和银行收付款。
- 银行日记账和银行流水对账。
- 查询资金余额。

### 常见路径

```text
财务会计
  -> 出纳管理 / 资金管理
  -> 收款 / 付款 / 银行存款日记账 / 银行对账
```

### 操作步骤：银行收款

1. 根据银行流水确认到账。
2. 判断是客户回款、其他收款、保证金还是内部往来。
3. 录入或关联收款单。
4. 选择正确银行账号和结算方式。
5. 审核收款单并生成凭证。
6. 与应收单核销。
7. 银行日记账和银行流水对账。

### 操作步骤：银行付款

1. 根据审批通过的付款申请生成付款单。
2. 检查供应商、收款账号、金额、用途。
3. 审核付款单。
4. 完成网银或银企直连付款。
5. 生成付款凭证。
6. 与应付单核销。
7. 银行日记账和银行流水对账。

### 检查点

- 银行账号是否正确。
- 收付款组织是否正确。
- 结算方式是否正确。
- 是否重复收款或重复付款。
- 是否及时生成凭证和核销。
- 银行日记账是否每日对账。

## 五、固定资产日常操作

### 适用场景

- 新购资产入账。
- 资产变更、调拨、处置。
- 每月计提折旧。
- 固定资产台账和总账对账。

### 常见路径

```text
资产管理
  -> 固定资产
  -> 日常管理 / 折旧管理 / 账务处理 / 结账
```

### 操作步骤：新增资产卡片

1. 确认采购、验收和发票资料。
2. 新增资产卡片。
3. 录入资产名称、资产类别、使用部门、使用人、存放地点。
4. 录入原值、税额、启用日期、折旧方法、预计使用期间、残值率。
5. 审核资产卡片。
6. 生成资产入账凭证。
7. 查固定资产台账和总账固定资产科目。

### 操作步骤：月末折旧

1. 确认当月新增、变更、处置资产已经处理完成。
2. 执行折旧计算。
3. 检查折旧清单。
4. 生成折旧凭证。
5. 审核并过账折旧凭证。
6. 查询折旧费用科目、累计折旧科目。
7. 固定资产台账和总账对账。

### 常见异常

| 现象 | 可能原因 | 处理 |
| --- | --- | --- |
| 折旧金额不对 | 原值、残值率、年限、方法错误 | 回查资产卡片 |
| 折旧凭证生成失败 | 费用科目或部门维度缺失 | 查资产类别和模板 |
| 资产台账和总账不一致 | 入账或折旧凭证未过账 | 查凭证状态 |
| 资产无法结账 | 当期变更、处置或折旧未完成 | 查固定资产结账检查 |

## 六、报表查询和对账操作

### 常用报表

| 报表 | 用途 | 查询前先确认 |
| --- | --- | --- |
| 科目余额表 | 看科目期初、本期发生、期末余额 | 组织、账簿、期间、币别 |
| 明细账 | 看科目每笔发生 | 科目、期间、是否含未过账 |
| 应收账龄 | 看客户欠款和逾期 | 客户、组织、核销状态 |
| 应付账龄 | 看供应商未付款 | 供应商、组织、核销状态 |
| 银行日记账 | 看银行账号流水 | 银行账号、期间 |
| 固定资产台账 | 看资产原值、折旧、净值 | 会计政策、期间 |
| 财务报表 | 出资产负债表、利润表、现金流量表 | 报表公式、期间、是否结转 |

### 对账操作顺序

```text
固定查询条件
  -> 查子模块明细
  -> 查凭证状态
  -> 查科目余额表
  -> 查报表公式
  -> 输出差异清单
  -> 分类处理
```

### 对账差异分类

| 差异类型 | 典型表现 | 处理 |
| --- | --- | --- |
| 未生成凭证 | 子模块有单据，总账没有发生额 | 生成凭证并过账 |
| 未过账凭证 | 凭证存在但余额表不体现 | 审核过账 |
| 手工凭证 | 总账有发生额，子模块无单据 | 查手工凭证依据 |
| 期初差异 | 从第一期开始就不一致 | 回查初始化 |
| 查询条件差异 | 两边数据都对但口径不同 | 固定组织、账簿、期间、币别 |

## 七、每日、每周、每月操作节奏

### 每日

- 审核当天业务单据。
- 处理当天收款和付款。
- 生成并审核过账当天业务凭证。
- 查询未生成凭证清单。
- 出纳核对银行流水。

### 每周

- 检查应收账龄和重点逾期客户。
- 检查应付到期付款计划。
- 抽查手工凭证和大额凭证。
- 检查新增客户、供应商、科目、银行账号。
- 对重点科目做余额复核。

### 每月

- 完成应收、应付、出纳、固定资产、库存、成本等模块结账。
- 完成凭证审核、过账和损益结转。
- 完成固定资产折旧。
- 完成报表出具和对账。
- 整理月结证据包。

## 八、操作留痕建议

财务操作要能追溯。建议保存：

- 单据编号和凭证号。
- 报错截图或报错原文。
- 处理前后金额。
- 处理人、审核人、过账人。
- 反审核、反过账、反结账原因。
- 月结检查表和差异处理表。

## 九、常用快捷检查清单

### 凭证生成前

- [ ] 源单已审核。
- [ ] 业务日期正确。
- [ ] 会计期间开放。
- [ ] 凭证模板启用。
- [ ] 科目映射完整。
- [ ] 辅助核算完整。
- [ ] 金额和税额不为空。

### 凭证过账前

- [ ] 借贷平衡。
- [ ] 科目正确。
- [ ] 辅助核算完整。
- [ ] 摘要可读。
- [ ] 附件齐全。
- [ ] 凭证日期正确。
- [ ] 来源单据可追溯。

### 月结前

- [ ] 未审核单据清零。
- [ ] 未生成凭证单据清零。
- [ ] 未审核凭证清零。
- [ ] 未过账凭证清零。
- [ ] 应收应付核销检查完成。
- [ ] 固定资产折旧完成。
- [ ] 子模块和总账对账完成。
- [ ] 报表公式检查完成。
