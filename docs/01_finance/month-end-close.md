## 适用范围

适用于金蝶财务月末结账、跨模块检查、结账失败排查和月结 SOP 制定。

## 核心原则

月结不是最后一步才开始，而是每个子模块都要完成“单据完整、审核完成、凭证生成、凭证过账、账实核对”之后，总账才能顺利结账。

## 推荐月结顺序

不同企业流程会有差异，常见顺序如下：

1. 供应链单据检查：采购、销售、库存单据是否审核。
2. 库存关账：检查未审核单据、盘点、负库存、锁库或预留异常。
3. 应付款结账：检查应付单、其他应付单、付款单、付款退款单。
4. 出纳结账：检查收付款单、退款单、现金存取单、银行转账单。
5. 固定资产结账：检查资产新增、变更、处置、折旧、凭证。
6. 存货核算和成本核算：确认出入库成本和成本计算结果。
7. 应收款结账：检查应收单、其他应收单、收款单、收款退款单。
8. 总账处理：凭证审核过账、期末调汇、自动转账、结转损益。
9. 总账结账：账簿进入下一期间。
10. 报表出具：资产负债表、利润表、现金流量表和管理报表。

## 总账结账前检查

- 本期凭证是否全部审核、过账。
- 相关业务系统是否已经结账。
- 业务单据是否全部生成凭证。
- 生成的业务凭证是否全部过账。
- 外币业务是否完成期末调汇。
- 自动转账凭证是否生成、审核、过账。
- 损益类科目是否完成结转。

## 常见结账阻塞

| 提示类型 | 排查方向 |
| --- | --- |
| 存在未审核单据 | 按模块和单据类型查未审核数据 |
| 存在未生成凭证单据 | 检查智能会计平台规则和凭证生成日志 |
| 存在未过账凭证 | 查凭证状态、审核状态、期间状态 |
| 子模块未结账 | 回到应收、应付、出纳、资产、库存等模块结账 |
| 余额不平或对账不一致 | 检查期初、手工凭证、辅助核算、跨期单据 |

## 月结 SOP 模板

| 顺序 | 模块 | 负责人 | 检查内容 | 完成标记 |
| --- | --- | --- | --- | --- |
| 1 | 库存 | 供应链/仓库 | 单据审核、盘点、负库存 |  |
| 2 | 应付 | 应付会计 | 应付单、付款单、核销、凭证 |  |
| 3 | 出纳 | 出纳 | 银行对账、收付款单、凭证 |  |
| 4 | 固定资产 | 资产会计 | 折旧、处置、资产凭证 |  |
| 5 | 应收 | 应收会计 | 应收单、收款单、核销、凭证 |  |
| 6 | 总账 | 总账会计 | 凭证、调汇、转账、损益结转 |  |
| 7 | 报表 | 财务经理 | 报表检查、差异说明 |  |
