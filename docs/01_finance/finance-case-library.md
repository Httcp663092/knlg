这里收集金蝶财务模块中最常遇到的真实工作场景。每个案例都按“现象 -> 排查 -> 根因 -> 处理 -> 预防”来写，方便复用。

## 案例 1：应收账龄显示逾期，但客户已经付款

### 现象

客户 A 已经付款，收款单也已经审核，但应收账龄分析表仍显示这笔应收逾期。

### 排查

1. 查应收单是否仍有未核销金额。
2. 查收款单是否已审核。
3. 查收款单是否生成凭证并过账。
4. 查收款单是否和应收单完成核销。
5. 查客户、结算客户、币别、组织是否一致。

### 常见根因

收款单只是记录了回款，没有和具体应收单核销。

### 处理

执行收款核销，将收款单匹配到对应应收单；重新查询账龄表。

### 预防

把“收款后核销”作为应收会计每日检查动作，不要等月末集中处理。

## 案例 2：应收余额和总账应收账款不一致

### 现象

应收模块客户余额合计为 100 万，总账应收账款科目余额为 102 万。

### 排查

1. 统一账簿、期间、组织、币别。
2. 查应收模块是否有未生成凭证单据。
3. 查应收凭证是否全部过账。
4. 查总账应收账款明细账是否存在手工凭证。
5. 按客户辅助核算和应收模块客户余额逐一比对。

### 常见根因

总账手工录入了一张借记应收账款 2 万的调整凭证，没有在应收模块形成单据。

### 处理

如果调整真实存在，要在对账底稿中列明；如果要求子模块和总账一致，应补录应收调整单或其他应收单，再冲回手工凭证。

### 预防

规定应收账款、预收账款等往来科目原则上不得直接手工制证。

## 案例 3：付款已经做了，应付账龄仍未减少

### 现象

付款单已审核，银行也扣款了，但供应商应付账龄仍显示未付款。

### 排查

1. 查付款单是否生成凭证。
2. 查付款单是否和应付单核销。
3. 查付款供应商和应付供应商是否一致。
4. 查币别和结算组织是否一致。

### 常见根因

付款单没有核销对应应付单。

### 处理

执行付款核销；如果供应商或币别错误，需要按流程反审核或红冲后重做。

### 预防

付款完成后，当天检查付款单未核销金额。

## 案例 4：业务单据不能生成凭证，提示科目为空

### 现象

应收单或应付单审核后生成凭证失败，提示科目为空或科目取不到。

### 排查

1. 查凭证模板是否启用。
2. 查模板适用组织、账簿、单据类型是否命中。
3. 查科目来源：固定科目、物料、客户、供应商、费用项目或业务类型。
4. 查对应基础资料是否维护了科目。
5. 查凭证模板条件是否把当前业务排除。

### 常见根因

凭证模板通过物料类别或费用项目取科目，但基础资料上没有维护对应科目。

### 处理

补充基础资料科目映射，或调整凭证模板取科目规则；重新生成凭证。

### 预防

新增物料、费用项目、客户、供应商类型时同步检查科目映射。

## 案例 5：凭证生成成功，但辅助核算维度缺失

### 现象

凭证生成了，但应收账款科目没有客户维度，或费用科目没有部门维度。

### 排查

1. 查科目是否启用辅助核算。
2. 查凭证模板是否配置辅助核算取值。
3. 查业务单据字段是否有客户、供应商、部门、项目。
4. 查是否因为合并凭证导致维度被合并或丢失。

### 常见根因

科目启用了客户辅助核算，但凭证模板没有把单据客户带到凭证分录。

### 处理

调整凭证模板辅助核算取值；删除或冲销错误凭证，重新生成。

### 预防

凭证模板测试时，不只看借贷金额，还要逐行看辅助核算维度。

## 案例 6：固定资产折旧金额不对

### 现象

某资产本月折旧金额明显偏大或偏小。

### 排查

1. 查资产原值、累计折旧、净值。
2. 查预计使用年限、残值率、折旧方法。
3. 查开始使用日期和启用期间。
4. 查本月是否有资产变更。
5. 查折旧政策是否设置当月新增当月折旧。

### 常见根因

资产卡片录入时预计使用期间或开始使用日期错误。

### 处理

按权限和流程做资产变更或反审核调整；重新计提折旧；检查折旧凭证。

### 预防

资产卡片上线或新增时，必须由资产会计和总账会计复核折旧要素。

## 案例 7：固定资产和总账固定资产科目不一致

### 现象

固定资产模块资产原值合计与总账固定资产科目余额不一致。

### 排查

1. 查资产新增、变更、处置单据是否全部审核。
2. 查资产入账、折旧、处置凭证是否全部生成。
3. 查相关凭证是否全部过账。
4. 查总账是否有手工凭证影响固定资产或累计折旧科目。
5. 查期初资产卡片是否和总账期初一致。

### 常见根因

本期资产处置单已审核，但处置凭证未生成或未过账。

### 处理

生成并过账处置凭证；重新核对资产报表和总账。

### 预防

固定资产结账前必须执行“资产台账 vs 总账科目”核对。

## 案例 8：银行对账不平

### 现象

银行对账后仍有差异，银行余额和企业账余额不同。

### 排查

1. 查银行账号、币别、期间是否一致。
2. 查银行流水是否重复导入。
3. 查收付款单是否漏录。
4. 查手续费、利息、银行扣款是否已入账。
5. 查未达账项是否有合理说明。

### 常见根因

银行已扣手续费，但金蝶未录付款或费用单据。

### 处理

补录手续费相关单据并生成凭证；重新对账。

### 预防

银行流水导入后，先处理手续费、利息、自动扣款等非业务收付款。

## 案例 9：月结提示存在未过账凭证

### 现象

总账结账时提示本期存在未过账凭证。

### 排查

1. 凭证列表按期间过滤。
2. 状态筛选未过账。
3. 查未过账原因：未审核、凭证错误、期间问题、权限问题。
4. 查是否来自业务单据，是否应回源单处理。

### 常见根因

业务凭证已生成但主管未审核，所以无法过账。

### 处理

完成凭证审核后过账；如果凭证错误，先修正来源单据或模板。

### 预防

月结前每天检查未审核、未过账凭证清单。

## 案例 10：结转损益后利润表仍不对

### 现象

已生成结转损益凭证，但利润表本月数或本年累计数异常。

### 排查

1. 查损益类科目是否全部过账。
2. 查新增损益科目是否纳入结转方案。
3. 查利润表公式是否包含新增科目。
4. 查结转凭证是否审核过账。
5. 查报表取数期间和账簿是否正确。

### 常见根因

新增费用科目进入凭证和科目余额表，但没有加入利润表公式。

### 处理

维护报表公式，重新计算报表；必要时调整结转方案。

### 预防

新增科目必须同步检查：凭证模板、结转规则、报表公式、辅助核算。

## 案例 11：外币应收期末调汇结果异常

### 现象

外币客户应收余额调汇后，汇兑损益金额明显不合理。

### 排查

1. 查外币科目是否启用外币核算。
2. 查期末汇率是否维护正确。
3. 查调汇方案是否包含该科目。
4. 查客户应收原币余额是否正确。
5. 查是否存在未过账的外币凭证。

### 常见根因

期末汇率维护错误，或调汇方案遗漏部分外币科目。

### 处理

修正汇率或调汇方案，重新生成调汇凭证并复核。

### 预防

期末调汇前先导出外币科目余额和汇率表，由财务复核。

## 案例 12：报表资产负债表不平

### 现象

资产负债表资产合计不等于负债和所有者权益合计。

### 排查

1. 查总账凭证是否全部过账。
2. 查报表公式是否覆盖全部科目。
3. 查新增科目是否遗漏。
4. 查重分类规则是否正确。
5. 查本年利润、未分配利润、损益结转是否正确。

### 常见根因

新增科目没有加入资产负债表公式。

### 处理

维护报表公式并重新计算；同时检查利润表和现金流量表是否也受影响。

### 预防

科目新增流程中增加“报表公式检查”步骤。
