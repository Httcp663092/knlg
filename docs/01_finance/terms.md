## 基础对象

| 术语 | 解释 |
| --- | --- |
| 核算组织 | 承担会计核算责任的组织 |
| 会计核算体系 | 组织会计核算关系和规则的体系 |
| 会计政策 | 决定会计日历、本位币、科目体系等核算口径 |
| 账簿 | 一套财务数据的归集口径 |
| 主账簿 | 某核算组织在同一核算体系内的主要账簿 |
| 副账簿 | 用于不同准则、不同币别或管理口径的辅助账簿 |
| 科目表 | 会计科目的结构体系 |
| 科目 | 记录会计业务的分类项目 |
| 核算维度 | 科目之外的辅助核算口径，如客户、供应商、部门、项目 |

## 总账

| 术语 | 解释 |
| --- | --- |
| 凭证 | 会计分录记录 |
| 凭证审核 | 对凭证合法性和准确性进行确认 |
| 出纳复核 | 对涉及现金银行科目的凭证进行资金侧复核 |
| 过账 | 将凭证登记到账簿 |
| 期末调汇 | 对外币科目进行汇兑损益处理 |
| 自动转账 | 按模板自动生成期末转账凭证 |
| 结转损益 | 将损益类科目余额结转到本年利润 |
| 反结账 | 将账簿期间回退到上一期间 |

## 应收应付

| 术语 | 解释 |
| --- | --- |
| 应收单 | 确认客户债权的财务单据 |
| 其他应收单 | 非销售业务形成的应收单据 |
| 收款单 | 记录客户收款 |
| 应付单 | 确认供应商债务的财务单据 |
| 其他应付单 | 非采购业务形成的应付单据 |
| 付款单 | 记录对外付款 |
| 核销 | 将应收与收款、应付与付款等进行匹配 |
| 账龄 | 按到期或发生时间分析往来余额 |

## 固定资产

| 术语 | 解释 |
| --- | --- |
| 资产卡片 | 固定资产的基础记录 |
| 折旧方法 | 计算折旧的规则 |
| 折旧政策 | 影响折旧起止、残值和计提规则的政策 |
| 资产变更 | 对资产使用部门、位置、折旧等信息调整 |
| 资产处置 | 资产报废、出售、转让等退出处理 |
