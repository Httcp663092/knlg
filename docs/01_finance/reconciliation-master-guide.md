对账是财务系统可信的关键。金蝶中业务模块、财务模块、总账、报表之间环环相扣，只要其中一个环节状态、期间、口径、凭证或维度不一致，就会出现“业务有数、总账没数”“报表不平”“应收应付对不上”等问题。

## 一、对账总原则

- 先固定口径，再比较数字。
- 先看模块报表，再看总账。
- 先查凭证生成和过账，再怀疑系统数据。
- 先按明细定位，再做汇总结论。
- 差异必须分类，不要只写一个差额。

## 二、对账口径

每次对账前确认：

| 口径 | 示例 |
| --- | --- |
| 组织 | A 公司、B 工厂 |
| 账簿 | 主账簿、副账簿 |
| 期间 | 2026-06 |
| 币别 | 原币、本位币 |
| 状态 | 已审核、已过账 |
| 科目 | 应收账款、应付账款、存货 |
| 维度 | 客户、供应商、部门、项目 |

## 三、应收对账

```text
销售出库 / 应收单
  -> 收款单
  -> 核销
  -> 应收余额
  -> 应收凭证
  -> 总账应收科目
```

检查：

- 应收单是否审核。
- 收款是否核销。
- 应收凭证是否生成。
- 凭证是否过账。
- 总账是否有手工应收凭证。
- 客户辅助核算是否完整。

差异分类：

| 差异 | 判断 |
| --- | --- |
| 未生成凭证 | 应收有余额，总账没有 |
| 未过账 | 凭证存在但总账余额不体现 |
| 手工凭证 | 总账有，应收模块无来源 |
| 核销差异 | 应收和收款未完全核销 |
| 期初差异 | 上线期初明细和总账不一致 |

## 四、应付对账

```text
采购入库 / 应付单
  -> 付款单
  -> 核销
  -> 应付余额
  -> 应付凭证
  -> 总账应付科目
```

检查：

- 应付单是否审核。
- 付款是否核销。
- 暂估是否冲回。
- 发票是否匹配。
- 凭证是否生成和过账。
- 供应商辅助核算是否完整。

## 五、库存和存货对账

库存数量和存货金额要分开看。

```text
库存单据
  -> 即时库存数量
  -> 库存明细账
  -> 存货核算金额
  -> 存货凭证
  -> 总账存货科目
```

检查：

- 库存单据是否全部审核。
- 是否存在负库存。
- 存货核算是否完成。
- 出库成本是否为 0 或异常。
- 存货凭证是否生成和过账。
- 总账是否有手工存货凭证。

常见差异：

- 库存有数量无金额。
- 存货核算未完成。
- 凭证未生成。
- 凭证未过账。
- 手工调整存货科目。
- 期初库存和总账不一致。

## 六、固定资产对账

```text
资产卡片
  -> 折旧计提
  -> 资产变动 / 处置
  -> 资产凭证
  -> 总账固定资产和累计折旧科目
```

检查：

- 卡片原值汇总 = 固定资产科目余额。
- 卡片累计折旧汇总 = 累计折旧科目余额。
- 本期折旧 = 折旧费用凭证。
- 处置资产不再计提折旧。
- 资产部门和费用部门一致。

## 七、资金对账

```text
收付款单
  -> 银行日记账
  -> 银行流水
  -> 银行对账
  -> 总账银行科目
```

检查：

- 收付款单是否审核。
- 是否生成凭证。
- 凭证是否过账。
- 银行流水是否认领。
- 未达账项是否合理。
- 内部转账是否重复统计现金流。

## 八、报表对账

报表不是单独对账对象，它要和总账对齐。

检查：

- 报表期间和账簿。
- 报表公式科目范围。
- 新增科目是否加入公式。
- 凭证是否全部过账。
- 结转损益是否完成。
- 重分类公式是否正确。

## 九、月结对账顺序

建议顺序：

```text
1 库存数量对账
2 存货核算金额对账
3 应收对账
4 应付对账
5 固定资产对账
6 出纳资金对账
7 总账试算平衡
8 报表检查
9 现金流量表检查
```

不要先总账结账再回头处理子模块差异。

## 十、差异表模板

| 模块 | 对账对象 | 子模块金额 | 总账金额 | 差异 | 差异原因 | 处理人 | 状态 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 应收 | 客户 A | 100,000 | 90,000 | 10,000 | 凭证未过账 | 总账会计 | 待处理 |
| 存货 | 原材料 | 500,000 | 520,000 | -20,000 | 手工凭证 | 成本会计 | 待确认 |

## 十一、实际案例：存货和总账差异

现象：

- 存货核算余额 1,000,000。
- 总账存货科目余额 1,050,000。
- 差异 50,000。

排查：

1. 查存货凭证生成情况，全部生成。
2. 查凭证过账状态，全部过账。
3. 查总账存货科目明细，发现一张手工凭证借记存货 50,000。
4. 询问财务，原来是临时调整供应商差异。
5. 该调整没有业务来源，导致存货和总账不一致。

处理：

- 冲回手工存货凭证。
- 通过存货调整单或正确业务流程处理。
- 建立存货科目手工凭证审批控制。

## 十二、实际案例：应收账龄和总账不一致

现象：

- 应收账龄客户 B 余额 200,000。
- 总账客户 B 应收余额 180,000。

排查：

1. 查客户 B 应收明细。
2. 找到一张 20,000 应收单未生成凭证。
3. 查凭证模板，发现该单据类型未配置。
4. 补模板后生成凭证并过账。

结论：

- 差异来自业务凭证未生成，不是客户余额错误。

## 十三、对账检查清单

- [ ] 组织、账簿、期间、币别一致。
- [ ] 子模块单据已审核。
- [ ] 凭证已生成。
- [ ] 凭证已过账。
- [ ] 辅助核算完整。
- [ ] 手工凭证已识别。
- [ ] 期初差异已确认。
- [ ] 差异按原因分类。
- [ ] 处理后再次复核。
- [ ] 月结对账证据已归档。
