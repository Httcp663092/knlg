财务软件最终不是为了录凭证，而是为了把企业经营结果变成可理解、可分析、可决策的报表。资产负债表看企业“有什么、欠什么、归谁”，利润表看“赚不赚钱”，现金流量表看“钱从哪里来、到哪里去”。三张表必须联动看，单看任何一张都容易误判。

## 一、三大报表关系

```text
资产负债表：时点
利润表：期间经营结果
现金流量表：期间现金变化
```

核心关系：

```text
资产 = 负债 + 所有者权益

利润增加会影响所有者权益

现金流量表解释货币资金的变化
```

如果利润表显示盈利，但现金流量表显示经营现金流持续为负，就要重点关注应收账款、存货和回款。

## 二、资产负债表

资产负债表反映某一时点企业的财务状况。

### 资产

| 类别 | 示例 | 关注点 |
| --- | --- | --- |
| 流动资产 | 货币资金、应收账款、存货 | 变现能力和周转 |
| 非流动资产 | 固定资产、无形资产、长期投资 | 长期投入和折旧摊销 |

### 负债

| 类别 | 示例 | 关注点 |
| --- | --- | --- |
| 流动负债 | 短期借款、应付账款、应交税费 | 短期偿债压力 |
| 非流动负债 | 长期借款、长期应付款 | 长期融资结构 |

### 所有者权益

- 实收资本。
- 资本公积。
- 盈余公积。
- 未分配利润。

## 三、利润表

利润表反映一定期间企业经营成果。

```text
营业收入
- 营业成本
- 税金及附加
- 销售费用
- 管理费用
- 研发费用
- 财务费用
+ 其他收益
+ 投资收益
- 所得税费用
= 净利润
```

分析重点：

- 收入是否增长。
- 毛利率是否稳定。
- 费用率是否异常。
- 财务费用是否增加。
- 净利润是否和经营现金流匹配。

## 四、现金流量表

现金流量表解释现金变化。

重点看：

- 经营活动现金流是否为正。
- 投资活动现金流是否用于扩张。
- 筹资活动现金流是否依赖借款。
- 净利润和经营现金流差异原因。

## 五、三张表联动看

### 利润高但现金少

可能原因：

- 应收账款增加。
- 存货积压。
- 预付款增加。
- 收入确认早于收款。

金蝶里查：

- 应收账龄表。
- 客户余额表。
- 库存周转。
- 销售出库和收款核销。

### 现金多但利润低

可能原因：

- 收到借款。
- 收回历史应收。
- 收到投资款。
- 延迟付款给供应商。

金蝶里查：

- 银行流水。
- 收款单来源。
- 借款凭证。
- 应付账龄。

### 收入增长但利润下降

可能原因：

- 毛利下降。
- 费用增长。
- 折扣增加。
- 成本核算异常。
- 低毛利客户或产品占比提高。

金蝶里查：

- 销售收入明细。
- 销售成本明细。
- 产品毛利分析。
- 费用部门分析。

## 六、常用财务指标

### 偿债能力

```text
流动比率 = 流动资产 / 流动负债
速动比率 = (流动资产 - 存货) / 流动负债
资产负债率 = 负债总额 / 资产总额
```

用途：

- 判断短期和长期偿债压力。
- 银行、投资人、管理层常看。

### 营运能力

```text
应收账款周转天数 = 平均应收账款 / 营业收入 * 天数
存货周转天数 = 平均存货 / 营业成本 * 天数
应付账款周转天数 = 平均应付账款 / 采购或成本 * 天数
```

用途：

- 看回款速度。
- 看库存占用。
- 看供应商付款周期。

### 盈利能力

```text
毛利率 = (营业收入 - 营业成本) / 营业收入
净利率 = 净利润 / 营业收入
费用率 = 期间费用 / 营业收入
ROA = 净利润 / 平均总资产
ROE = 净利润 / 平均净资产
```

用途：

- 看企业赚钱能力。
- 比较产品、客户、部门、项目。

## 七、金蝶里怎么取数

| 分析 | 常用数据 |
| --- | --- |
| 应收分析 | 应收账龄表、客户余额、收款核销 |
| 应付分析 | 应付账龄表、供应商余额、付款计划 |
| 库存分析 | 库存收发存、存货核算、库龄 |
| 毛利分析 | 销售收入、销售成本、产品分类 |
| 费用分析 | 科目余额表、部门辅助余额 |
| 现金分析 | 银行日记账、收付款单、现金流量表 |
| 资产分析 | 固定资产清单、折旧明细 |

## 八、分析维度

常用维度：

- 组织。
- 部门。
- 客户。
- 供应商。
- 产品。
- 物料分类。
- 项目。
- 业务员。
- 区域。
- 期间。

指标必须和维度结合才有管理价值。例如毛利率按公司看没问题，但如果要改善经营，就要进一步按产品、客户、区域、销售员看。

## 九、实际案例：利润增长但经营现金流为负

现象：

- 6 月净利润 100 万。
- 经营活动现金流 -200 万。

排查：

1. 查应收账款，发现本月新增应收 300 万。
2. 查销售收款，发现大客户账期延长。
3. 查存货，发现备货增加 120 万。
4. 查应付，发现供应商付款提前。

结论：

- 企业账面盈利，但现金被客户欠款和库存占用。

金蝶动作：

- 应收账龄按客户跟踪。
- 建客户信用和回款计划。
- 建库存周转报表。
- 采购付款与销售回款联动管理。

## 十、实际案例：毛利率突然下降

现象：

- 5 月毛利率 32%。
- 6 月毛利率 18%。

排查：

1. 按产品分类分析毛利率。
2. 发现某产品成本单价异常升高。
3. 查存货核算，发现 6 月存在负库存出库。
4. 查采购入库，发现入库成本补录较晚。
5. 重算存货成本后毛利率恢复。

结论：

- 毛利下降不是销售问题，而是库存成本核算异常。

## 十一、报表分析常见误区

| 误区 | 正确理解 |
| --- | --- |
| 利润等于现金 | 利润按权责发生制，现金按收付实现 |
| 收入增长一定好 | 要看回款、毛利、费用和风险 |
| 库存越多越安全 | 库存占用现金，可能跌价 |
| 应付越多越好 | 拖欠供应商可能影响供应链 |
| 报表数字不对就是公式错 | 也可能是凭证、期间、维度、过账问题 |

## 十二、分析检查清单

- [ ] 报表期间和组织一致。
- [ ] 凭证已过账。
- [ ] 新增科目已进入报表公式。
- [ ] 应收、应付、库存、资产和总账已对账。
- [ ] 分析指标有清晰公式。
- [ ] 指标维度和管理动作对应。
- [ ] 异常波动能追到单据和凭证。
- [ ] 分析结论能转化为行动。
