## 二、组织与账簿

| 检查项 | 标准 | 状态 |
| --- | --- | --- |
| 核算组织清单完整 | 所有独立核算主体已确认 |  |
| 账簿清单完整 | 每个核算组织账簿已确认 |  |

| 检查项 | 标准 | 状态 |
| --- | --- | --- |
| 科目表已确认 | 编码、名称、级次、方向完整 |  |

## 九、上线验收

| 验收项 | 标准 | 状态 |
| --- | --- | --- |
| 期初借贷平衡 | 总账期初平衡 |  |
| 子模块期初一致 | 应收、应付、资产、资金与总账一致 |  |
| 典型业务跑通 | 销售应收、采购应付、收付款、折旧、月结 |  |
| 凭证生成正确 | 分录、科目、辅助核算、金额正确 |  |
| 报表出具正确 | 资产负债表、利润表等公式正确 |  |
| 月结演练通过 | 子模块和总账能顺利结账 |  |

## 十、上线后一个月重点观察

- 是否有大量凭证生成失败。
- 是否有应收应付核销滞后。
- 是否有手工凭证直接影响往来科目。
- 是否有报表公式遗漏新增科目。
- 是否有月结检查重复出现同类错误。
- 是否有业务人员绕开标准流程录单。
