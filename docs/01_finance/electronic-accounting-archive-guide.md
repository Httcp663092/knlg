电子会计档案用于把业务单据、发票、合同、银行回单、凭证、报表、审批记录和附件形成可检索、可追溯、可审计的档案。金蝶项目中，电子档案不是“把附件传上去”这么简单，它要和业务流程、凭证、权限、归档规则、保管期限和审计查询结合。

## 一、适用范围

- 业务单据附件管理。
- 发票和凭证影像归档。
- 合同、验收单、回单等附件归档。
- 财务凭证和原始单据关联。
- 审计抽查和档案检索。
- 电子会计档案系统集成。

## 二、档案对象

| 档案对象 | 示例 |
| --- | --- |
| 业务单据 | 采购订单、销售订单、费用报销、付款申请 |
| 财务凭证 | 总账凭证、业务生成凭证 |
| 发票 | 销售发票、采购发票、电子发票 |
| 银行回单 | 收款回单、付款回单 |
| 合同 | 采购合同、销售合同、服务合同 |
| 审批记录 | 工作流审批意见和时间 |
| 报表 | 月结报表、科目余额表、财务报表 |

## 三、归档原则

- 一笔业务能追溯到原始附件。
- 一张凭证能反查来源单据。
- 一个附件能看到归属组织、期间、单据和责任人。
- 归档后不能随意删除或替换。
- 档案权限要和业务权限、财务权限匹配。
- 归档文件要有备份和保管期限。

## 四、业务到凭证的档案链

```text
业务申请
  -> 合同或订单
  -> 收发货或服务验收
  -> 发票
  -> 收付款
  -> 凭证
  -> 报表
  -> 电子档案
```

一个完整档案包最好包含：

- 业务单据。
- 审批记录。
- 合同或订单附件。
- 发票附件。
- 入库、出库或验收凭据。
- 银行回单。
- 凭证。

## 五、附件规范

### 文件命名

建议规则：

```text
组织_期间_单据类型_单据编号_附件类型
```

示例：

```text
A公司_2026-06_付款单_FKD0001_银行回单.pdf
```

### 附件类型

| 类型 | 示例 |
| --- | --- |
| 合同 | 合同扫描件、补充协议 |
| 发票 | 电子发票、发票验真结果 |
| 回单 | 银行回单、收款证明 |
| 验收 | 入库单、验收单、签收单 |
| 审批 | 线下审批单、说明 |
| 其他 | 邮件、截图、客户确认函 |

## 六、凭证归档

凭证归档要关注：

- 凭证号。
- 会计期间。
- 账簿。
- 制单人、审核人、过账状态。
- 来源单据。
- 附件张数。
- 原始单据链接。

凭证如果来自业务单据，归档时应保留来源关系，不要只保存凭证 PDF。

## 七、电子发票归档

电子发票归档字段：

- 发票号码。
- 发票代码。
- 开票日期。
- 购买方。
- 销售方。
- 税号。
- 金额。
- 税额。
- 价税合计。
- 发票状态。
- 版式文件。
- 验真结果。

注意：

- 红字发票要关联蓝字发票。
- 发票状态变化要同步。
- 发票附件不能重复归档。

## 八、权限和审计

档案权限要分层：

| 角色 | 权限 |
| --- | --- |
| 业务人员 | 查看本人或本部门单据附件 |
| 财务人员 | 查看财务相关单据、凭证和发票 |
| 审计人员 | 按授权范围查询档案 |
| 档案管理员 | 归档、移交、保管 |
| 系统管理员 | 系统维护，不应随意查看敏感附件 |

敏感附件：

- 银行账号。
- 合同金额。
- 薪酬费用。
- 税务资料。
- 客户隐私信息。

## 九、归档流程

```text
业务单据完成
  -> 附件完整性检查
  -> 凭证生成和过账
  -> 档案规则匹配
  -> 自动或手工归档
  -> 档案验收
  -> 存储和备份
  -> 审计查询
```

归档时点可以是：

- 单据审核后。
- 凭证过账后。
- 月结后。
- 项目验收后。

## 十、常见问题

| 问题 | 原因 | 处理 |
| --- | --- | --- |
| 凭证找不到原始单据 | 来源关系未保留或手工凭证无附件 | 要求凭证附件和来源说明 |
| 附件重复上传 | 缺少唯一规则 | 用单据编号和附件类型控制 |
| 审计查不到档案 | 权限或归档状态问题 | 查档案权限和归档日志 |
| 电子发票和应付不匹配 | 发票号码、供应商或金额不一致 | 做发票匹配规则 |
| 附件丢失 | 存储或备份不规范 | 建附件备份和恢复演练 |

## 十一、实际案例

### 案例：付款凭证审计追溯

审计要求：

- 抽查 6 月付款 50 万以上凭证。
- 要看到付款申请、审批、合同、发票、银行回单。

系统应支持：

1. 从总账凭证查付款凭证。
2. 反查付款单。
3. 查看付款申请审批记录。
4. 查看合同和发票附件。
5. 查看银行回单。
6. 导出审计档案包。

如果只能看到凭证，没有原始附件，审计追溯就不完整。
