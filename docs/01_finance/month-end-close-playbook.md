这篇是财务月结的操作型手册，按时间线和失败排查来写。

## 目标

月结完成后应满足：

- 子模块已经结账。
- 业务单据已完整生成凭证。
- 凭证已审核和过账。
- 往来、资金、资产、存货和总账能对上。
- 资产负债表、利润表、现金流量表或管理报表能出具。
- 所有差异有说明。

## 时间线建议

| 时间 | 工作 | 负责人 | 输出 |
| --- | --- | --- | --- |
| D-5 | 提醒业务部门清理未审核单据 | 财务经理/业务负责人 | 未审核清单 |
| D-3 | 检查采购、销售、库存关键单据 | 供应链会计 | 单据完成率 |
| D-2 | 应收应付收付款核销 | 往来会计 | 往来余额表 |
| D-1 | 银行对账和现金盘点 | 出纳 | 资金对账表 |
| D | 固定资产折旧、存货核算、成本核算 | 资产/成本会计 | 折旧和成本结果 |
| D+1 | 生成并审核业务凭证 | 总账会计 | 凭证检查表 |
| D+2 | 期末调汇、自动转账、结转损益 | 总账会计 | 期末凭证 |
| D+3 | 总账结账和报表出具 | 财务经理 | 月结包 |

## 子模块检查

### 应收

- 应收单、其他应收单是否全部审核。
- 收款单、收款退款单是否全部审核。
- 本期应收和收款是否全部生成凭证。
- 收款是否完成核销。
- 应收账龄是否有异常大额逾期。
- 应收余额是否和总账核对。

### 应付

- 应付单、其他应付单是否全部审核。
- 付款单、付款退款单是否全部审核。
- 本期应付和付款是否全部生成凭证。
- 付款是否完成核销。
- 到期债务是否有异常。
- 应付余额是否和总账核对。

### 出纳

- 银行对账单是否导入完整。
- 银行日记账是否完整。
- 银行对账是否完成。
- 未达账项是否有说明。
- 涉及现金银行科目的凭证是否复核、过账。

### 固定资产

- 本期新增资产是否建卡。
- 本期资产变更是否审核。
- 本期资产处置是否审核。
- 折旧是否计提。
- 折旧凭证和处置凭证是否生成并过账。
- 资产台账是否和总账核对。

### 总账

- 本期凭证是否全部审核。
- 本期凭证是否全部过账。
- 外币业务是否完成期末调汇。
- 自动转账凭证是否生成。
- 损益是否结转。
- 结账检查是否通过。

## 月结失败排查路线

### 提示存在未审核单据

处理：

1. 记录提示模块和单据类型。
2. 到对应单据列表按期间和审核状态过滤。
3. 判断是正常业务未处理，还是误录单据。
4. 正常业务补审核，误录单据作废或删除。
5. 重新执行结账检查。

### 提示存在未生成凭证单据

处理：

1. 打开凭证生成或凭证生成情况查询。
2. 按账簿、期间、单据类型过滤。
3. 找到未生成的源单。
4. 如果生成失败，看失败原因。
5. 修正凭证模板、科目映射、辅助核算或金额规则。
6. 重新生成凭证并审核过账。

### 提示存在未过账凭证

处理：

1. 查询凭证列表，过滤未过账。
2. 确认是否未审核、期间关闭、凭证错误或权限问题。
3. 审核并过账。
4. 如果凭证错误，回源单或凭证模板修正。

### 提示子模块未结账

处理：

1. 看提示模块。
2. 回该模块执行期末处理或结账检查。
3. 处理该模块未审核、未生成凭证、未核销、未对账问题。
4. 子模块结账后再回总账。

## 月结包建议

每个月结账完成后，建议保存以下资料：

- 科目余额表。
- 凭证明细或凭证汇总。
- 应收余额表和账龄表。
- 应付余额表和账龄表。
- 银行对账结果和未达账项说明。
- 固定资产折旧表和资产余额表。
- 月结检查表。
- 资产负债表、利润表、现金流量表。
- 差异说明和调整凭证清单。

## 不建议的做法

- 为了结账临时删除或跳过错误业务单据。
- 子模块没对平就强行在总账手工调整。
- 新增科目后不检查报表公式。
- 业务单据跨期但不记录说明。
- 结账后不保存差异底稿。

## 月结完成判断

| 判断项 | 标准 |
| --- | --- |
| 单据 | 本期关键业务单据无未审核异常 |
| 凭证 | 本期凭证无未审核、无未过账异常 |
| 子模块 | 应收、应付、出纳、固定资产等已结账 |
| 对账 | 往来、资金、资产与总账差异为 0 或有说明 |
| 报表 | 主要报表能出具且勾稽关系正确 |
| 留痕 | 调整事项、差异事项有底稿 |
