## 适用范围

适用于财务报表出具、科目余额核对、往来对账、资产对账和月结后的数据检查。

## 报表分类

| 类别 | 常见报表 | 用途 |
| --- | --- | --- |
| 总账报表 | 科目余额表、明细账、总分类账、辅助账 | 核对科目和凭证 |
| 往来报表 | 应收明细、应付明细、客户/供应商对账单 | 核对客户和供应商往来 |
| 账龄报表 | 应收账龄、应付账龄、到期债权/债务 | 分析回款和付款风险 |
| 资金报表 | 银行日记账、现金日记账、银行余额 | 核对资金 |
| 资产报表 | 资产清单、折旧明细、资产变动 | 核对资产台账 |
| 财务报表 | 资产负债表、利润表、现金流量表 | 对外和管理报表 |

## 对账思路

### 子模块对总账

核心问题：子模块余额是否等于总账科目余额。

常见核对：

- 应收款余额 vs 应收账款科目余额。
- 应付款余额 vs 应付账款科目余额。
- 银行/现金余额 vs 银行存款/库存现金科目余额。
- 固定资产原值和累计折旧 vs 总账资产类科目。

### 明细对汇总

核心问题：明细单据或辅助账是否能加总到汇总数。

检查条件要保持一致：

- 账簿
- 期间
- 币别
- 组织
- 科目范围
- 客户/供应商/部门/项目等辅助维度
- 是否包含未过账凭证

### 报表对科目

核心问题：报表项目公式是否正确。

排查顺序：

1. 看报表项目公式。
2. 查公式对应科目余额。
3. 查科目是否有新增或调整但公式未维护。
4. 查借贷方向和重分类规则。
5. 查本期是否存在未过账凭证。

## 常见差异原因

- 业务单据未生成凭证。
- 凭证已生成但未审核或未过账。
- 手工凭证直接影响往来或资产科目。
- 期初余额和明细期初不一致。
- 查询条件不一致。
- 科目公式、辅助核算或核算维度配置不一致。
- 跨期单据或跨期凭证造成期间差异。

## 个人检查清单

- [ ] 报表期间和账簿正确。
- [ ] 所有凭证已审核过账。
- [ ] 子模块已完成结账。
- [ ] 应收应付与总账核对。
- [ ] 现金银行与总账核对。
- [ ] 固定资产与总账核对。
- [ ] 损益类科目已结转。
- [ ] 报表公式覆盖新增科目。
- [ ] 差异有说明和责任人。
