# 固定资产

## 适用范围

适用于固定资产启用、资产卡片、资产入账、变更、处置、折旧、盘点和固定资产月结。

## 核心目标

固定资产模块用于管理资产从入账到处置的全生命周期，并将折旧、处置等结果同步到总账。

## 启用流程

1. 设置固定资产参数。
2. 启用固定资产系统，设置启用年度和期间。
3. 维护基础资料。
4. 录入初始化卡片。
5. 处理日常资产业务。
6. 执行期末处理。

## 基础资料

常见基础资料包括：

- 资产类别
- 资产状态
- 变动方式
- 折旧方法
- 资产位置
- 折旧政策
- 会计政策

## 日常业务

| 业务 | 说明 |
| --- | --- |
| 资产申请 | 资产购置或使用申请 |
| 资产入账 | 建立资产卡片，确认资产入账 |
| 资产处置 | 报废、出售、转让等退出处理 |
| 资产变更 | 部门、使用人、折旧政策等变更 |
| 资产借用 | 临时借出或借入 |
| 资产领用 | 资产投入使用 |
| 批量变更 | 批量调整资产信息 |

## 期末处理

固定资产期末通常关注：

- 资产盘点是否完成。
- 本期新增、变更、处置是否审核。
- 折旧是否计提。
- 折旧或处置凭证是否生成并过账。
- 固定资产系统是否结账。

## 常见问题

### 折旧金额不符合预期

检查资产原值、累计折旧、净值、预计使用期间、残值率、折旧方法、开始使用日期、是否当期新增当期折旧。

### 资产卡片不能结账

检查本期资产变动、处置、折旧、盘点、凭证生成和过账状态。

### 固定资产和总账不一致

检查资产入账、折旧、处置相关凭证是否完整生成并过账；排查是否存在手工凭证影响固定资产科目。
