## 推荐内容

- Web API
- OpenAPI
- 第三方系统对接
- 银企直连
- 税务和发票平台
- 电子会计档案
- 数据同步和中间库

## 财务集成关注点

- 外部单据进入金蝶后是否形成正式财务单据。
- 金额、税额、币别、汇率和核算维度是否完整。
- 接口失败是否有重试、幂等和异常处理。
- 财务凭证生成是否可追溯到外部来源。
