# knlg

个人知识库，用来沉淀项目经验、领域资料、方法论、问题记录和可复用模板。

## 快速入口

- [知识库首页](docs/README.md)
- [知识地图](docs/knowledge-map.md)
- [记录规范](docs/conventions.md)
- [临时收集箱](docs/inbox/README.md)
- [长期领域](docs/areas/README.md)
- [项目沉淀](docs/projects/README.md)
- [资料库](docs/resources/README.md)
- [归档](docs/archive/README.md)
- [笔记模板](docs/templates/note.md)

## 使用方式

1. 新想法、待整理资料先放到 `docs/inbox/`。
2. 稳定维护的主题放到 `docs/areas/`。
3. 和具体目标、交付物相关的内容放到 `docs/projects/`。
4. 外部资料、链接、书籍、课程和工具说明放到 `docs/resources/`。
5. 过期但仍有参考价值的内容移到 `docs/archive/`。

```text
  projects/              项目沉淀

  archive/               归档

```

## 维护节奏

- 每天：把零散记录先放入 `docs/inbox/`。
- 每周：整理 inbox，把内容移动到合适分类。
- 每月：更新 `docs/knowledge-map.md`，删除重复内容，归档过期笔记。
